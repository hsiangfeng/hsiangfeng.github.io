常見的狀況

通常來講有一種狀況會語系包失敗，就是在更新 VS Code 的時候會發生，通常有兩種解法

  • 重新啟動一整個 VS Code，這一招在 macOS 有效，而 Window 則是無效。
  • 移除語系包再重新安裝，這一招基本上兩個作業系統都可以解決。
