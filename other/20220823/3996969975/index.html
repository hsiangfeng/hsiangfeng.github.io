前言

Discord 已經越來越紅了，甚至許多公司都陸續改使用 Discord 來取代原有的 Slack，但我們有時候會希望可以更高度客製化自己的 Discord 讓自己在使用 Discord 的更舒適，因此接著這一篇就會來記錄一下如何優化、客製化自己的 Discord。

BetterDiscord 安裝

Discord 如果要客製化的話，我們必須安裝一套軟體叫做「BetterDiscord」，這一套軟體就可以幫助我們高度客製化與優化 Discord，而 BetterDiscord 本身也有開源在 GitHub，所以如果覺得有疑慮的人可以自己上去看原始碼。

(Discord 本身是使用 Electron 所製作的。)

接著讓我們來記錄一下安裝教學（底下安裝全程以 Mac 為舉例），安裝之前建議關閉 Discord 軟體，避免出現奇怪問題

首先到 BetterDiscord 官方網站首頁點擊下載軟體

下載之後解壓縮後會看到一個「BetterDiscord.app」雙擊

你可能會出現這個訊息「無法打開「BetterDiscord.app」，因為無法驗證開發者。」，因為 Mac 對於沒有驗證過的軟體是相對嚴格的

這時候按下「取消」打開 Mac 的「系統偏好設定」，在上方搜尋「安全性與隱私權」

接著切換到「一般」底下會有一個「強制打開」將 BetterDiscord.app 強制打開(記得先點一下左下鎖頭)

這時候會問你是不是真的要打開，你就點「打開」

打開之後就可以準備安裝 BetterDiscord 了，只需要一直下一步看到下面選取 Discord 就對了

這樣子 Discord 的擴充功能就安裝完畢了，你可以打開 Discord 的「個人設定」頁面，左側滾到最下方就可以看到多一個區塊是「BETTERDISCORD」

客製化主題

接著你可以進入點這個 連結 進入到主題商店下載主題

看到你喜歡的主題就點進去，然後點 Download 就對了

這邊舉例 ClearVision

下載好之後你會取得一包 CSS 檔案(ClearVision_v6.theme.css)，接著打開 Discord 個人設定 => 佈景主題 => 開啟佈景主題資料夾

接著將你剛剛下載的 CSS 檔案丟進去

最後回到 Discord 畫面你就可以看到主題出現在畫面上了，然後只需要打開開關就可以套用主題囉~

推薦套件

由於安裝套件的流程跟安裝主題是一樣的，所以就不從頭介紹了，只需要到這個 網址 就可以下載了

那麼這邊我也只介紹幾個我有使用的 Plugin，因為套件裝太多反而很容易開啟 Discord 很慢，因此安裝套件的時候會建議慎選比較好

  • 增強程式碼片段
    • 簡介說明：主要讓程式碼片段可以高亮
  • 查看使用者 Discord 註冊時間
    • 簡介說明：由於我是 Discord 管理人，所以有時候會需要避免一些傳廣告訊息的帳戶進來，因此這功能對我來講就可以驗證該使用者是何時註冊的
  • 圖片支援觸碰版放大
    • 簡介說明：有時候我們在看別人圖片時會希望可以放大看，那個這個套件就可以幫助我們放大與縮寫
  • 拼字檢查
    • 簡介說明：這是一個非常好用的套件，如果你是一名工程師的話，相信你應該很熟悉，當你單字打錯的時候會有一個提示告知你拼字有錯
  • Spotify 控制中心
    • 簡介說明：簡單來講就是你可以在 Discord 裡面控制 Spotify，算是給不想要一直切換視窗的人一個小方便
  • Google 翻譯
    • 簡介說明：如果你有加入一些國外社群，那麼這套件可以幫你直接翻譯語言，可以說是人在江湖走 Google 翻譯要有
  • 雙擊編輯
    • 簡介說明：你可以透過點擊兩下訊息直接進入編輯文字模式

那麼由於套件非常多，所以我只列出我有在使用的套件哩
