前言

Raycast 是一款可以完全取代 Spotlight 的 Mac 應用程式，為什麼會說它可以「完成取代」呢？因為它真的非常強大，而且還可以自訂功能，讓它可以做到你想要的功能，至於有多強大，就讓我們來看看吧！

Raycast 安裝

首先一開始請你先進入到 Raycast 的官方網站下載 Raycast（目前僅支援 macOS），下載後打開 Raycast 可以看到底下的歡迎畫面

接著點一下「Skip setup」，這樣就可以直接進入到 Raycast 的主畫面了

非常簡單吧？而且整體也非常像 Spotlight 呢！

更改 Raycast 快捷鍵

首先預設 Raycast 呼叫方式是 Option + Space，可是我們這邊是想要取代 Spotlight，所以我們要把它改成 Command + Space，這樣才能更方便的使用，而且也不會跟 Spotlight 有衝突

首先請打開你的「系統偏好設定」，然後點選「鍵盤」或者「搜尋 Spotlight」，接著點選「鍵盤快速鍵」

接著找到「Spotlight」，然後把「顯示 Spotlight 搜尋」的快速鍵取消打勾，並按下完成

接著回到 Raycast，點選左下角找到「Setting」進入設定頁面

基本上預設進來會是「General」，所以這邊找到 Raycast Hotkey，然後把它改成 Command + Space 就可以囉~

Raycast 基本功能

接下來是針對 Raycast 的基本功能介紹，其實 Raycast 由於與 Spotlight 非常相似，所以你也可以在上面搜尋你要開啟的 APP，或者當計算機 XD

所以我們可以知道 Raycast 本身就擁有 Spotlight 原有的功能，因此你也可以把 Raycast 當作進階版的 Spotlight 來使用，所以就接下來介紹一些我認為滿有用的技巧吧！

設定快捷鍵

首先我自己非常常使用 Visual Studio Code，所以這邊我們就可以針對常用的應用程式設置快速開啟的快捷鍵

首先打開 Raycast 進入 Setting，然後點選「Extensions」，接著搜尋「Visual Studio Code」

接著在 Alias 輸入「vscode」，接下來你就可以在 Raycast 輸入「vscode」就可以直接開啟 Visual Studio Code，而且你也可以看到畫面上有顯示「vscode」的快捷鍵

設定搜尋引擎

Raycasy 另一個很強大的功能就是可以直接在上面搜尋你要的資料，只需要在 Raycast 上面輸入「Search Google」，然後接著輸入你要搜尋的內容，就可以按下 Enter 進行搜尋

如果覺得這招很麻煩，你可以替它設定一個快捷鍵，首先打開 Raycast 進入 Setting，然後點選「Extensions」，接著搜尋「Search Google」，接著我改成了「/」

這樣未來我需要搜尋時，只需要輸入「/」，然後接著輸入你要搜尋的內容，就可以按下 Enter 進行搜尋

那麼開啟的瀏覽器會是依照你電腦預設的為主。

剪貼簿

Raycast 本身也有提供剪貼簿功能，方便你回頭去查看剪貼簿的內容，而且也可以直接貼上剪貼簿的內容，只需要在 Raycast 上面輸入「Clipboard」，就可以看到剪貼簿的內容

這個功能可以避免你明明複製了 A 內容，但不小心跑去複製了 B 內容導致原有的內容被覆蓋，可以說是相當的方便。

設定擴充套件

以上這些都是 Raycast 本身的功能，但 Raycast 也可以透過擴充套件來擴充功能，例如我們可以安裝 Google 翻譯的擴充套件。

首先先進入 Raycast Store 網站，接著點一下搜尋框，然後輸入「Google Translate」

接著按下「Install Extension」，就可以安裝 Google 翻譯的擴充套件

接下來會跳出歡迎視窗

接著會進入安裝「Google Translate」的畫面，只需要按下「Install」就可以安裝完成

安裝完畢後我們就要來設置一下翻譯語言，在 Raycast 上面輸入「Google Translate」，接著按下「Enter」，就可以看到翻譯的語言設定

接著在按下「Enter」，就會進入翻譯介面，接著我們就可以輸入要翻譯的內容囉

（謎之聲：你才雷。）

當然你也可以替它設置快捷鍵，以我來講我就設置成了 gt。

結尾

那麼 Raycast 其實有非常多的功能，所以真要介紹還真的有點介紹不完，所以其餘 Raycast 的功能就讓你自己去探索囉。

我的快捷設定

最後也提供我目前有設置的快捷鍵，方便你參考~

  • gt：Google 翻譯
  • vscode：Visual Studio Code
  • chrome：Google Chrome
  • /：搜尋引擎
  • ff：Firefox
  • cal：行事曆
  • clip：剪貼簿
  • todo：Todoist
  • note：Notion
  • sp：Spotify
