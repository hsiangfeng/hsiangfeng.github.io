前言

Notion 是一款非常簡潔且非常好用的筆記軟體，好用到我直接拋棄我使用了將近快五年的 EverNote，所以這一篇來介紹一下 Notion 有什麼樣的魅力可以讓我拋棄 EverNote 吧。

EverNote

Image

EverNote 與 Notion 一樣都是一款雲端筆記軟體，可以完整紀錄一整個網頁，你可以將整個網頁複製貼上到 EverNote 都沒問題任何，而且也支援跨平台，不論是手機、Windows 到 Mac 都可以使用，在早期時對我來講是真的非常方便好用的筆記軟體。

但是在後來 EverNote 開始收費後我就開始覺得有點不是那麼好用，這邊僅列出我個人觀點

跨裝置同步只能兩台

跨裝置同步

由於我是一名工程師，所以我會使用到兩台電腦跟一個手機(公司電腦與家裡電腦)，說真的光有這個限制我就覺得非常雞肋，每次換電腦使用 EverNote 就必須先解除另一台電腦，而且一個月只能解除三次（印象）。

一個月只能上傳 60 MB

上載配額

雖然這個功能並不是非常重要，但是有一次我剛好這個月上傳了三四次約 10 MB 左右的檔案，偏偏剛好這些都是與筆記相關，額外找地方放也不是很好，所以變成這個限制有點麻煩，當然後來上傳檔案的部分我都盡可能改放其他空間。

每一個記事大小限制在 25 MB

Image

由於有些時候我會需要上傳圖片，這時候這個 MB 就很容易超出...QQ 而且這個 MB 還會關聯到前面的「一個月只能上傳 60 MB」，如果你有三個記事本都剛好在 25 MB 那麼你的記事就無法同步於雲端上。

編輯器不支援 MD 格式

由於我本身是一名工程師，早期在使用 EverNote 時並沒有很流行 Markdown 格式，但是後來近幾年 Markdown 格式越來越火紅，EverNote 也一直都沒有支援 Markdown，一直停留在原始的 Word 編輯器模式，雖然一直有聽說 EverNote 即將支援 Markdown 格式撰寫。

Ps.我記得我前陣子好像有看到有一個國家的 EverNote 已經可以支援 Markdown 格式撰寫，但國際版還沒支援。

最後這邊額外提供一個工具可以讓你在 EverNote 上撰寫 Markdown 格式

在EverNote 上撰寫 Markdown 格式

此外 EverNote 在撰寫程式碼也相當的不靈活也慢慢演變成我想找替代的原因之一。

有一段時間沒更新

EverNote 最後一次更新的穩定版本時間大概是在 2017/09/02(參考維基百科)，所以認真說起來，也真的有一段時間沒更新了。

以上差不多就是為什麼我慢慢拋棄 EverNote 的原因，但 EverNote 還是有它好用的地方，只是它已經不是那麼符合我的平常需求而已。

Notion

接下來講講本篇的重點，也就是 Notion。

Notion 除了支援 Markdown 撰寫之外它的 UI 也是相當簡潔漂亮，熱愛簡潔的人真的會愛上

圖源網路

更不用說有相當強大的預設 Template，這邊後面會再介紹。

在這邊先講一下 Notion 的幾個主要特色

  • Markdown 格式
  • Block 區塊
  • 針對程式碼高亮處理
  • 非常優秀的 Template 模板
  • 一鍵快速分享筆記
  • 豐富的權限管理
  • 可快速匯入其他筆記軟體資料，例如 EverNote，但需注意匯入後的格式問題
  • 版本控制
  • 客製化筆記網址
  • 教育帳號個人版免費 Personal

接下來我們就來慢慢介紹吧。

Markdown

前面有講到 Markdown 是這幾年非常火紅的撰寫格式，除了 Line 開始支援之外 Facebook、Messages 等都有支援 Markdown 格式，透過這些社群工具就可以知道 Markdown 有多火紅，以我個人來講我也相當的喜歡，在我撰寫這一篇文章的同時也是使用 Markdown 撰寫的，如果你對於 Markdown 撰寫方式不清楚，那麼你也可以參考我先前寫的 Markdown 基礎與入門教學手冊。

如果你想跟我一樣架設部落格的話，也可以考慮參考這一篇 試著學 Hexo 系列。

Block

Notion 主要是以區塊為單位，不論是文字或是圖片等等，都是作為一個 block 為單位，其中你也可以在這些 Block 中放入其他 Notion pages

Block

此外這些 Block 也可以讓你任意拖曳排序，所以可以說是相當的方便。

那就來慢慢介紹 Block 有哪些吧。

Basic Block(基礎區塊元素)

基本上光 Block 就可以分為以下

  • 文字區塊 (Text Block)
  • 頁面區塊 (Page Block)
  • 勾選區塊 (To-do List Block)
  • 標題區塊 (Heading1~3 Block)
  • 項目符號區塊 (Bulleted list Block)
  • 數字清單區塊 (Numbeered list Block)
  • 展開/收合區塊 (Toggle list Block)
  • 引用區塊 (Quote Block)
  • 分隔線區塊 (Divider Block)
  • 頁面連結區塊 (Lint To Page Block)
  • 標注區塊 (Callout Block)

Block

Inline(行內元素)

Block 功能中其實還有 inline 區塊，也就是行內區塊，如果你是一名網頁工程師，那麼應該會覺得很神似網頁的 Block inline 觀念。

  • 標註其他人 (Mention a person)
  • 標註其他頁面 (Mention a page)
  • 設定時間或提醒 (Date or Reminder)

inline

DataBase (資料區塊)

接下來我就會比較少一一介紹了，因為功能實在太多，僅列出來而已

  • 行內表格 (Table - line)
  • 行內看板 (Board - line)
  • 行內畫廊 (Gallery - line)
  • 行內清單 (List - line)
  • 行內日曆 (Calendar - line)
  • 整頁表格 (Table - full page)
  • 整頁看板 (Board - full page)
  • 整頁畫廊 (Gallery - full page)
  • 整頁清單 (List - full page)
  • 整頁日曆 (Calendar - full page)
  • 建立現有資料連結 (Create linked database)

基本上裡面 Gallery 就非常有特色，可以用作為一個任務管理系統。

Media (多媒體區塊)

這區塊我就不多作介紹，基本上就是圖片、影片以及網頁及音訊等等，其實就是你想得到的多媒體功能。

Embeds (嵌入區塊)

Embeds 可以嵌入的功能滿多的，例如 Google Drive、GitHub Gist 等等都可以，可以說是相當方便。

Advanced Blocks(進階/高級區塊)

基本上這區塊是比較少使用到，我自己也沒使用過，所以就不介紹了，只是這 Block 滿特別是可以寫數學方程式。

以上算是比較常見的 Block，因為 Notion 本身有提供相當多的 Block 因此還是建議你加入去玩玩比較好。

Template

Notion 本身有提供相當多非常好用的模板，基本上你只需要點一下左側功能列的 Template 就可以看到了

Template

它的模板非常多，所以我這邊也不一一介紹，讓你自己去挑選，當你有喜歡的只需要按「Use This Template」就會自動生成在左側功能列，可以說是非常非常的方便，而我目前僅只有使用 Notes & Drafts 這個模板。

而且你也可以上傳你做好的 Template 給別人使用唷！

每一個筆記都是一個 WorkSpace

基本上 Notion 每一個筆記都是一個空間，而你可以建立無限個空間，但是這邊要注意每一個空間都是獨立的，例如你在 A 空間付費，但不能套用到 B 空間上。

但是如果是團隊使用的話是真的滿方便的，你可以透過切換確保每一個筆記都是獨立的，舉例來講你可以建立一個工作專用的空間以及私人專用的空間。

此外你也可以針對該空間給予一個獨立的 Url 固定分享給他人，甚至邀請朋友成為嘉賓一起共享。

Notion 教育帳號免費 Personal

Notion 早期本身計費方式是使用 Block 來做單位，免費帳號最多只能使用 1000 個 block，但是後來 Notion 改版後不再限制 1000 個 block。

因此目前 Notion 免費版與個人 Pro 版本最大的差異點在於以下

  • 只能上傳 5MB 的檔案
  • 無法追朔三十天的歷史紀錄
  • Guest 最多五人（若有共筆習慣的人，這個會最明顯）

但是如果你是教師或是學生持有教育帳號的話，就可以無視上面限制直接升級成個人專業版

方案

而轉換的方式非常簡單，點一下左側功能列上的「Settings & Members」接下來點「Upgrade」滾到下方有一個「Students & educators」接下來按一下「Get free Education plan」驗證目前登入的帳號是否為學術帳號，當驗證成功就會自動變成 Personal 啦~

所以在這邊會建議註冊帳號一開始就註冊學術帳號會比較好。

ps.偷偷說基本上學術帳號可以將每一個 workspace 都升級成 Personal。

Notion 缺點

在前面講了一推 EverNote 的不符合需求的問題，所以這邊也列一些 Notion 的一些問題

  • 無中文介面
    • 據說未來會支援
  • 功能太多且較複雜，所以學習曲線較高
  • 多人使用單價較高，多人使用是以人頭為計算，一個人頭每月 8 美金。
  • 沒有二次驗證功能
  • 單價較高，主要是以 workspace 為計價，所以若有多個 workspace 就必須付多倍金額，而 EverNote 則是以一個帳號買斷制。

關於二次驗證功能這邊說明一下 Notion 的機制上是每次登入都會發送一個隨機密碼到信箱，如果是採用 Google 登入，那麼就會建議你將該帳號上鎖二次驗證會比較好，雖然 Notion 是有提供密碼設置，當你忘記 Google 帳號密碼時還可以透過密碼登入，但是密碼登入就沒有二次驗證保護，所以會建議你不要設置密碼會比較好。

如果你已經設置了密碼，但你想移除密碼，可是你找不到或是看不到「Remove Password」的選項時，我會建議你建立一個新的 workspace，然後在該 workspace 下移除，其主要原因在於升級的 workspace 是不能移除已經設置的 Password，但是新空間若還沒有升級就可以。

最後我還是要強調一下避免被誤會，EverNote 並不是不好，只是已經開始不符合我的使用需求而導致我慢慢地拋棄轉向 Notion。

Notion 繁體中文化

這邊也提供一個小技巧，當然也有許多人還是看不慣英文版，所以建議你可以試著打開網頁版，然後點右鍵翻譯成中文就可以全中文啦~

註冊 Notion

最後如果你看完這一篇之後也想使用 Notion，不仿可以透過該連結註冊 Notion

若你是透過我的這個連結註冊可以額外獲得 10 美金可以使用的額度(官方說的在下面)
