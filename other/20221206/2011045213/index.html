дҪ еҸҜд»Ҙд№ҹеҸҜд»Ҙи«ӢжҲ‘гҖҢе–қдёҖжқҜе’–е•Ў(Donate)гҖҚгҖӮ

пҪң
ж»ҝеӨҡдәәйғҪжңғжһ¶иЁӯиҮӘе·ұзҡ„жҠҖиЎ“йғЁиҗҪж јдёҰж”ҫзҪ®еңЁ GitHub PagesгҖҒGitLab Pages дёҠпјҢз„¶еҫҢд№ҹжңғжғіиҰҒе®ўиЈҪеҢ–иҮӘе·ұзҡ„йғЁиҗҪж јз¶ІеқҖпјҢжүҖд»ҘйҖҷдёҖзҜҮе°Үжңғд»Ӣзҙ№еҰӮдҪ•е®ўиЈҪеҢ–иҮӘе·ұйғЁиҗҪж јз¶ІеқҖпјҢз„¶еҫҢеҫһйӣ¶й–Ӣе§ӢиіјиІ·з¶ІеқҖпјҢдёҖи·ҜеҲ°е°ҮеҺҹжң¬ GitHub Pages з¶ІеқҖжӣҝжҸӣжҲҗиҮӘе·ұиіјиІ·зҡ„з¶ІеқҖдёҰжҗӯдёҠе…ҚиІ»зҡ„ SSH CloudFlareпјҒ

еүҚиЁҖ

ж»ҝеӨҡдәәйғҪжңғжһ¶иЁӯиҮӘе·ұзҡ„жҠҖиЎ“йғЁиҗҪж јдёҰж”ҫзҪ®еңЁ GitHub PagesгҖҒGitLab Pages дёҠпјҢз„¶еҫҢд№ҹжңғжғіиҰҒе®ўиЈҪеҢ–иҮӘе·ұзҡ„йғЁиҗҪж јз¶ІеқҖпјҢжүҖд»ҘйҖҷдёҖзҜҮе°Үжңғд»Ӣзҙ№еҰӮдҪ•е®ўиЈҪеҢ–иҮӘе·ұйғЁиҗҪж јз¶ІеқҖпјҢз„¶еҫҢеҫһйӣ¶й–Ӣе§ӢиіјиІ·з¶ІеқҖпјҢдёҖи·ҜеҲ°е°ҮеҺҹжң¬ GitHub Pages з¶ІеқҖжӣҝжҸӣжҲҗиҮӘе·ұиіјиІ·зҡ„з¶ІеқҖдёҰжҗӯдёҠе…ҚиІ»зҡ„ SSH CloudFlareпјҒ

дҪҝз”Ёе…ҚиІ» Domain зҡ„е•ҸйЎҢ

дёҖй–Ӣе§Ӣе…ҲдҫҶиҒҠдёҖдёӢзӮәд»ҖйәјеҘҪеҘҪзҡ„е…ҚиІ» Url дёҚз”ЁйӮ„иҰҒиҠұйҢўзү№еҲҘеҺ»иІ·дёҖеҖӢ Domain дҫҶеҺ»дҪҝз”Ёзҡ„еҺҹеӣ гҖӮ

йҰ–е…ҲйҖҡеёёжҲ‘еҖ‘жңғе°ҮиҮӘе·ұзҡ„йғЁиҗҪж ј(жҠҖиЎ“йғЁиҗҪж ј)иЁ—з®ЎеңЁ GitHub Pages жҲ–жҳҜ GitLab Pages дёҠпјҢиҖҢе®ҳж–№жң¬иә«жңғдҫқж“ҡдҪ зҡ„еёіиҷҹз”ҹжҲҗдёҖзө„дҪ е°Ҳеұ¬ UrlпјҢеҒҮиЁӯдҪ зҡ„ GitHub еёіиҷҹжҳҜ гҖҢisraynotarrayгҖҚпјҢйӮЈйәје°ұжңғжҳҜ гҖҢisraynotarray.github.ioгҖҚпјҢиҖҢ GitLab еүҮжҳҜ гҖҢisraynotarray.gitlab.ioгҖҚпјҢиҖҢйҖҷдәӣ Url йҖҡеёёе°ұжңғжҳҜдҪ йғЁиҗҪж јзҡ„з¶ІеқҖжҲ–жҳҜдҪ дҪңе“Ғе„Іеӯҳеә«зҡ„з¶ІеқҖгҖӮ

йӮЈд»ҖйәјдёҚеҘҪе‘ўпјҹйҰ–е…ҲйҖҷдәӣ Url жң¬иә«жҳҜ GitHub жҲ–жҳҜ GitLab жҸҗдҫӣзҡ„пјҢеӣ жӯӨе°ұжңғжңүе№ҫеҖӢе•ҸйЎҢзҷјз”ҹпјҡ

  • жҸҗдҫӣжңҚеӢҷзҡ„е№іеҸ°еҮәдәӢпјҢдҪ зҡ„з¶Із«ҷе°ұжңғжҺӣжҺүпјҢеӣ зӮәз¶ІеқҖйҖЈдёҚдёҠ
  • жҸҗдҫӣжңҚеӢҷзҡ„е№іеҸ°еҸ–ж¶Ҳи©Іж©ҹеҲ¶пјҢдҪ иҫӣиӢҰйӨҠзҡ„ SEO е°ұжІ’дәҶ
  • з¶ІеқҖйҖҡеёёдёҚжңғеҫҲеҘҪиЁҳпјҢиҖҢдё”д№ҹдёҚеӨӘеҘҪзңӢ

е…¶дёӯжңҖйә»з…©зҡ„е°ұжҳҜгҖҢиҪүз§»йғЁиҗҪж јгҖҚпјҢеҰӮжһңдҪ жҳҜжғіиҰҒе°ҮйғЁиҗҪж јж”№иҪүз§»еҲ°иҮӘе·ұжә–еӮҷзҡ„ж©ҹеҷЁдёӯпјҢйӮЈйәјйҖҷдәӣ SEO жөҒйҮҸе°ұз„Ўжі•и·ҹи‘—дҪ иө°пјҢеӣ жӯӨе…ҚиІ»зҡ„ Url йҖҡеёёйғҪжҳҜз”ЁдҫҶеҒҡдҪңе“Ғе„Іеӯҳеә«жҲ–жҳҜз°Ўе–®зҡ„йғЁиҗҪж јпјҢиҖҢдёҚжҳҜжғіиҰҒй•·жңҹз¶ӯиӯ·зҡ„йғЁиҗҪж јпјҢеҰӮжһңдҪ жңүжғіиҰҒй•·жңҹз¶ӯиӯ·жҲ–еҲҶдә«дҪ зҡ„йғЁиҗҪж јпјҢйӮЈйәје°ұжңғе»әиӯ°дҪ еҺ»иІ·дёҖеҖӢ Domain дҫҶдҪҝз”ЁгҖӮ

зңӢеҲ°йҖҷйӮҠпјҢеҰӮжһңдҪ дёҚзҹҘйҒ“д»ҖйәјжҳҜ Domain д№ҹдёҚзҹҘйҒ“ Domain зҡ„зҹҘиӯҳпјҢдҪ еҸҜд»ҘеҸғиҖғжҲ‘е…ҲеүҚеҜ«зҡ„йҖҷдёҖзҜҮж–Үз« гҖӮ

Google Domain

жҺҘдёӢдҫҶдҪ еҸҜд»Ҙжү“й–ӢдҪ зҡ„зҖҸиҰҪеҷЁпјҢеңЁз¶ІеқҖеҲ—дёҠијёе…ҘгҖҢhttps://domains.google.com/registrar/searchгҖҚ(жҲ–иҖ…й»һйҖҷеҖӢйҖЈзөҗ)пјҢз„¶еҫҢдҪ е°ұжңғзңӢеҲ° Google Domain зҡ„й ҒйқўгҖӮ

жҺҘдёӢдҫҶе‘ўпјҹжҺҘдёӢдҫҶдҪ еҸҜд»ҘеңЁең–зүҮзҡ„з®ӯй ӯжүҖжҢҮиҷ•ијёе…ҘдҪ жғіиҰҒзҡ„ Domain

дҫӢеҰӮпјҡдҪ еҸҜд»Ҙи©Ұи‘—жҗңе°Ӣ гҖҢRayIsGoodгҖҚпјҢз•¶дҪ жҢүдёӢжҗңе°ӢеҫҢдёӢж–№е°ұжңғи·іеҮәзӣёй—ңзҡ„ DomainпјҢиҖҢдҪ еҸҜд»ҘйҒёж“ҮдҪ жғіиҰҒзҡ„ Domain дҫҶдҪҝз”Ё

жҢ‘йҒёзҡ„жҷӮеҖҷиҰҒзЁҚеҫ®жіЁж„ҸдёҖдёӢпјҢжҜҸдёҖеҖӢ Domain йғҪжңғжңүдёҖеҖӢеғ№ж јпјҢйҡЁи‘—е°ҫзўјзҡ„дёҚеҗҢеғ№ж јд№ҹжңғдёҚеҗҢпјҢиҖҢеғ№ж јзҡ„й«ҳдҪҺд№ҹжңғеҪұйҹҝеҲ° Domain зҡ„еғ№еҖјпјҢдҫӢеҰӮпјҡгҖҢ.comгҖҚзҡ„ Domain жңүдәӣжңғжҜ”ијғиІҙпјҢиҖҢгҖҢ.twгҖҚзҡ„ Domain жңүдәӣеүҮжңғжҜ”ијғдҫҝе®ңпјҢиҖҢдҪ еҸҜд»ҘеңЁең–зүҮзҡ„з®ӯй ӯжүҖжҢҮиҷ•зңӢеҲ° Domain зҡ„еғ№ж јгҖӮ

з•¶дҪ йҒёеҘҪдҪ иҰҒзҡ„ Domain + е°ҫзўјеҫҢпјҢдҪ д№ҹеҸҜд»ҘзңӢдёҖдёӢе®ғзөҰдҪ зҡ„е»әиӯ°пјҢдҫӢеҰӮпјҡйҖҷеҖӢз¶ІеҹҹеҘҪдёҚеҘҪиЁҳгҖҒеёёдёҚеёёиҰӢзӯүе„Әй»һпјҢйҖҷдәӣйғҪжҳҜиіјиІ· Domain зҡ„жҷӮеҖҷеҸҜд»ҘиҖғж…®зҡ„еӣ зҙ пјҢеӣ зӮәйҖҷдәӣеҸҜд»Ҙи®“дҪ зҡ„з¶Із«ҷжӣҙе®№жҳ“иў«дәәжүҫеҲ°гҖӮ

ж—©жңҹ Google Domain еҸӘиғҪеңЁзҫҺеңӢиіјиІ·пјҢдҪҶжҳҜзҸҫеңЁе·Із¶“еҸҜд»ҘеңЁеҸ°зҒЈиіјиІ·дәҶпјҢиҖҢдё”д№ҹжңүжҸҗдҫӣдёӯж–Үзҡ„д»ӢйқўпјҢжүҖд»ҘдҪ еҸҜд»Ҙж”ҫеҝғзҡ„дҪҝз”ЁпјҢдҪҶжҳҜеӣ зӮәжҲ‘еҸҰдёҖеҖӢеёіиҷҹжҳҜеңЁзҫҺеңӢиіјиІ·пјҢеӣ жӯӨжҺҘдёӢдҫҶжҲ‘жңғз”ЁзҫҺеңӢеёіиҷҹдҫҶзӨәзҜ„(йҮ‘йЎҚжңғйЎҜзӨәзҫҺйҮ‘жҳҜжӯЈеёёзҡ„гҖӮ)

йӮЈйәјжңүдёҖд»¶жіЁж„ҸдәӢй …иҰҒжҸҗйҶ’пјҢз•¶дҪ дҪҝз”ЁйҒёж“ҮдҪҝз”ЁзҫҺеңӢең°еҚҖиіјиІ·жҷӮпјҢдҪ зҡ„ Google еёіе–®е°Үжңғеј·еҲ¶з¶Ғе®ҡеңЁзҫҺеңӢпјҢд№ҹе°ұжҳҜиӘӘдҪ зҡ„ Google Play е•Ҷеә—е°ұжңғеј·еҲ¶з¶Ғе®ҡеңЁзҫҺеңӢпјҢеҰӮжһңдҪ жңүиҰҒйҖҒзҰ®зөҰжңӢеҸӢзҡ„и©ұе°ұжңғз„Ўжі•йҖҒзҰ®пјҢйҖҷеҖӢиҰҒзЁҚеҫ®жіЁж„ҸдёҖдёӢгҖӮ

йҖҸйҒҺ Google Domain иіјиІ·з¶ІеқҖ

йӮЈйәјеүҚйқўжҲ‘еҖ‘жҢ‘йҒёдёҖзө„ Domain дҫҶиіјиІ·пјҢжҺҘдёӢдҫҶжҲ‘еҖ‘е°ұдҫҶзңӢзңӢеҰӮдҪ•йҖҸйҒҺ Google Domain иіјиІ·з¶ІеқҖдёҰеҲҶдә«иіјиІ·жөҒзЁӢгҖӮ

йҰ–е…ҲеҒҮдҪҝжҲ‘еҖ‘зӣ®еүҚжҢ‘йҒёзҡ„ Domain жҳҜгҖҢisraynotarray.devгҖҚпјҢдҪ еҸҜд»Ҙе…Ҳе°Үе®ғеҠ е…ҘеҲ°иіјзү©и»Ҡ

жҺҘдёӢдҫҶйҖІе…Ҙиіјзү©и»ҠеҫҢеҸҜд»ҘзңӢеҲ°дёҖжҺЁжқұиҘҝ

йҖҷйӮҠжңүе№ҫд»¶дәӢжғ…иҰҒжіЁж„ҸвҖҰ

  • з¬¬дёҖд»¶дәӢжғ…жҳҜзӣ®еүҚжҲ‘еҖ‘иіјиІ·зҡ„з¶ІеқҖй җиЁӯзӢҖжіҒдёӢжҳҜжІ’жңүе®үе…ЁиіҮж–ҷеӮіијёеұӨ (SSL) жҶ‘иӯүзҡ„пјҢдҪҶжҳҜйҖҷеҖӢжҲ‘еҖ‘еҫҢйқўжңғд»Ӣзҙ№ Cloudflare дҫҶе№«жҲ‘еҖ‘и§ЈжұәпјҢжүҖд»ҘйҖҷйӮҠдёҚз”Ёж“”еҝғгҖӮ

жҺҘдёӢдҫҶе°ұжҳҜиҰҒиЁӯе®ҡ CloudFlare зҡ„ DNSпјҢйҖҷйӮҠжҲ‘еҖ‘иҰҒжҠҠеҺҹжң¬зҡ„ DNS иЁӯе®ҡж”№жҲҗ CloudFlare зҡ„ DNSпјҢйҖҷжЁЈжүҚиғҪи®“ CloudFlare дҫҶе№«еҝҷиҷ•зҗҶ SSL зҡ„е•ҸйЎҢи·ҹж“Ғжңүе®ғжӣҙеӨҡзҡ„е…ҚиІ»еҠ еҖјжңҚеӢҷпјҢжүҖд»ҘйҖҷйӮҠиҰҒж–°еўһдёҖдәӣ DNS иЁӯе®ҡпјҢеҸӘйңҖиҰҒй»һдёҖдёӢгҖҢж–°еўһзҙҖйҢ„гҖҚпјҢж¬„дҪҚеЎ«еҜ«жҲҗд»ҘдёӢең–зүҮзҡ„жЁЈеӯҗеҚіеҸҜ

жҜ”ијғйңҖиҰҒжіЁж„Ҹзҡ„жҳҜ CNAME еҗҚзЁұеҝ…й ҲзӮә www з„¶еҫҢе…§е®№иҰҒеЎ«дҪ зҡ„еҺҹе§Ӣ GitHub Pages з¶ІеқҖпјҢдҫӢеҰӮжҲ‘зҡ„з¶ІеқҖжҳҜ share-jack.github.ioпјҢжүҖд»ҘжҲ‘е°ұиҰҒеЎ« share-jack.github.ioгҖӮ

жңҖеҫҢйҖҷйӮҠ CloudFlare жңғи·ҹдҪ иӘӘиҰҒеҺ»ж”№дёҖдёӢдҪ зҡ„еҗҚзЁұдјәжңҚеҷЁ

дҪ жңғеңЁйҖҷеҖӢз•«йқўеҸ–еҫ—д»ҘдёӢе…©еҖӢеҗҚзЁұдјәжңҚеҷЁ

жҺҘдёӢдҫҶжү“й–Ӣ Google Domain еҲ°дҪ иіјиІ·зҡ„з¶ІеқҖ DNS дёӯпјҢдёҰйҒёж“ҮгҖҢиҮӘиЁӮеҗҚзЁұдјәжңҚеҷЁгҖҚ

жҺҘи‘—е°ҮеүӣеүӣйӮЈе…©еҖӢеҗҚзЁұдјәжңҚеҷЁеЎ«е…Ҙеә•дёӢж¬„дҪҚпјҢдёҰжҢүдёӢе„Іеӯҳ

жҺҘи‘—жҢүдёҖдёӢдёҠж–№гҖҢеҲҮжҸӣзӮәйҖҷдәӣиЁӯе®ҡгҖҚжҢүйҲ•жүҚжҸӣзңҹжӯЈе®ҢжҲҗе”·пјҒ

еҰӮжһңжңүеҲҮжҸӣжҲҗеҠҹпјҢеҹәжң¬дёҠдҪ жңғеҫ—еҲ°йҖҷдёҖеҖӢиЁҠжҒҜгҖҢйҖҷдәӣ DNS иЁӯе®ҡеқҮзӮәжңүж•ҲгҖӮи®Ҡжӣҙжңғз«ӢеҚізҷјеёғпјҢдҪҶеҸҜиғҪйңҖиҰҒдёҖдәӣжҷӮй–“жүҚиғҪз”ҹж•ҲгҖҚпјҢдёҰдё”еҺҹжң¬зҡ„жЁҷйЎҢжңғи®ҠжҲҗгҖҢиҮӘиЁӮеҗҚзЁұдјәжңҚеҷЁ(дҪҝз”Ёдёӯ)гҖҚпјҢйҖҷжЁЈеӯҗжүҚжңүзңҹжӯЈзҡ„еҘ—з”ЁжҲҗиҮӘиЁӮзҡ„гҖӮ

жңҖеҫҢеӣһеҲ° CloudFlare зҡ„з•«йқўпјҢй»һйҒёгҖҢе®ҢжҲҗпјҢжӘўжҹҘеҗҚзЁұдјәжңҚеҷЁгҖҚ

йҖҷжҷӮеҖҷдҪ жҮүи©Іжңғи·іеҲ°дёҖеҖӢгҖҢеҝ«йҖҹе…Ҙй–ҖжүӢеҶҠгҖҚпјҢдҪ еҸҜд»Ҙж–ҹй…ҢиҰҒдёҚиҰҒзңӢпјҢдёҚз„¶е°ұжҳҜжҢүдёӢгҖҢзЁҚеҫҢе®ҢжҲҗгҖҚпјҢйҖҷжЁЈе°ұеҸҜд»Ҙи·іеӣһ CloudFlare зҡ„з•«йқўдәҶпјҢйӮЈйәје…¶еҜҰеҲ°йҖҷйӮҠе°ұе·Із¶“е®ҢжҲҗе°ҮжҲ‘еҖ‘иҮӘе·ұзҡ„з¶ІеҹҹеҘ—з”ЁеҲ° CloudFlare зҡ„иЁӯе®ҡпјҢеҸӘжҳҜйҖҷйӮҠ CloudFlare жңғйңҖиҰҒдёҖй»һжҷӮй–“еҺ»жӘўжҹҘдҪ зҡ„еҗҚзЁұдјәжңҚеҷЁпјҢжүҖд»ҘдҪ еҸҜд»Ҙе…ҲеҺ»еҒҡе…¶д»–дәӢжғ…пјҢзӯүеҫ…дёҖж®өжҷӮй–“еҫҢеҶҚеӣһдҫҶзңӢзңӢгҖӮ
