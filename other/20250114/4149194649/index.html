前言

前陣子有朋友看到我開啟專案以及操作速度非常之快，同為使用 Raycast 朋友就很好奇我是不是有私藏什麼超好用套件，所以這邊就來分享一下我推薦的 Raycast 擴充套件，讓你的工作流程更強大！

Raycast 套件推薦

那麼 Raycast 我先前已經有寫文章介紹過，所以這邊就不再贅述，如果你還不知道 Raycast 是什麼，可以參考我之前的文章：來吧！用 Raycast 取代 Mac 內建 Spotlight 吧！

那麼底下除了簡單說明一下這些套件之外，我也會附上下載連結以及我的使用心得 Alias（快捷鍵） 給你參考哩

Raycast - TinyPNG

身為前端開發者，應該對於 TinyPNG 都不陌生，畢竟是一個非常老牌的圖片壓縮網站，而這個 Raycast 擴充套件就是可以讓你直接在 Raycast 上使用 TinyPNG 的服務，讓你可以更快速的壓縮圖片！

但是要使用之前你必須要先去申請 TinyPNG Token 而申請的教學文章你可以參考我先前的文章：一步一步帶你用 Node.js 上傳檔案到 Firebase Storage 之 壓縮圖片(TinyPNG)

你只需要觀看「壓縮圖片(TinyPNG)」章節即可。

那麼一個月有 500 次圖片壓縮的機會，應該對於一般使用者來說是非常足夠的～

下載連結：Raycast - TinyPNG

Alias：tinypng

Raycast - Kill Process

Kill Process 是一款專門殺除卡在 Mac 上的 Process 的 Raycast 擴充套件，如果你很常遇到「Mac Port xxx is already in use」這種情況，那麼這個套件就非常適合你！

你可以快速的找到卡住的 Process 並且殺除，讓被卡住的 Port 釋放出來。

下載連結：Raycast - Kill Process

Alias：kill

Raycast - Visual Studio Code

這是我工作以及下班有超過 8 成都必定會使用的套件，以往我們再開啟專案都需要透過 Finder 找到專案資料夾，然後再開啟 Visual Studio Code，但是你可以直接透過這個專案直接搜尋關鍵字，就可以直接開啟專案，非常方便！

下載連結：Raycast - Visual Studio Code

Alias：code .

Raycast - Google Translate

超級無敵懶得每次都要打開瀏覽器搜尋「Google 翻譯」，現在你可以直接透過 Raycast 來使用 Google 翻譯服務

下載連結：Raycast - Google Translate

Alias：gt

Raycast - Speedtest

網路不順？不知道是不是網路問題？通常我們都要打開瀏覽器搜尋「Speedtest」，而這個套件基本上就是幫你把網路速度測試工具放到 Raycast 上

下載連結：Raycast - Speedtest

Alias：speed

Raycast - Format JSON

身為前端開發者一定對於 JSON 不陌生，但是有時候會拿到一些非常雜亂的 JSON，這時候你就可以透過這個套件來幫你格式化 JSON，讓你可以更加清楚的看到 JSON 的結構～

下載連結：Raycast - Format JSON

Alias：json

Raycast - Base64

Base64 如果你不熟悉的話，我非常推薦你要閱讀我這一篇文章加密是什麼呢？為什麼要加密？關於加密的基本觀念，我希望你在使用之前要先知道 Base64 並不是拿來加密用，而是一種編碼方式。

使用這個套件好處在於可以快速幫助我們將文字轉換成 Base64 編碼或者是將 Base64 編碼轉換成文字

下載連結：Raycast - Base64

Raycast - JWT Decoder

JWT 是一種非常常見的 Token 驗證方式，而這個 JWT 是可以反向解析的，如果你好奇這運作過程，可以參考Day29-關於 JWT 驗證，這一篇文章我將會詳解 JWT 的運作方式

那麼這個套件目的就是幫我們將 JWT Token 解析成 JSON 格式，讓我們可以更加清楚的看到 Token 的內容與格式等

下載連結：Raycast - JWT Decoder

Alias：jwt

那麼以上就是我有使用且推薦的 Raycast 擴充套件，希望這些擴充套件可以大大提升你的工作效率唷～
