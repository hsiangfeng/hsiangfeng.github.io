博弈公司的工程師生活：揭開神秘面紗，機會與挑戰並存

前言

博弈產業是一個讓每個人充滿好奇的產業，因為它的神秘感及高薪水的特性，讓許多人對於這產業會感到特別的好奇，但是同時也讓許多人對於進入這個產業充滿的害怕與恐懼，為什麼呢？這一篇我就依照自己經歷以及身邊的朋友們的經驗來跟大家分享一下，希望能夠讓大家對於這個產業有更多的了解以及該如何挑選博弈產業的公司。

什麼是博弈產業

網路上如果你看人家討論在講菠菜的話，其實就是在講博弈，而菠菜是博彩公司的代名詞，簡單來講博弈就是與賭博有相關的產業。

那麼在博弈產業上班就一定違法嗎？如果是以台灣來講的話，基本上無論是在公開場合或在網路賭博都會涉犯賭博罪。

有趣的事情來了，我們都知道在台灣賭博是違法的，那為什麼博弈產業還是可以在台灣存在呢？我相信這是一個大家都很好奇的問題，但是這個問題牽扯的層面太多且太複雜，總結一句就是只要是在台灣，博弈公司都是遊走在灰色地帶（意旨不合法但也不違法），因此不管怎樣，博弈公司必定會有它的風險存在，只是你必須要去評估這個風險的大小而已。

那麼接下來就會依照我先前在博弈公司的經驗以及身邊朋友的經驗來跟大家分享。

Note

這一篇並不代表全部博弈產業，只是我個人的經驗以及詢問身邊朋友的經驗而已，因此並不代表全部博弈產業都是這樣。

博弈產業的技術及生活

基本上博弈生態的技術可大可小，以我先前待的公司來講主要技術是 AngularJS + Vue.js，而我朋友的公司則是 React、Vue.js 都有，因此以技術面來講其實滿看公司的，所以你可能會常聽到人家說

「博弈公司大多技術都很落後」

又或者是

「前端常見使用 AngularJS」

其實這些都算是一個刻板印象，因為博弈公司的技術迭代速度也非常快，所以也會有非常高的機會使用到新技術的，所以並不代表博弈公司的技術就一定是落後的。

那麼因為我在博弈產業的時間並沒有很長，因此我還特別問了待比較久的朋友分享一下，基本上博弈公產業的領域可以拆分為三大類，分別是

  • 內容
  • 平台
  • 金流

什麼是內容、平台與金流呢？簡單來講內容就是博弈遊戲相關的開發，平台則是入口網站，金流就不用多說了，就是處理金錢的部分。

你可能會好奇我曾經負責過什麼呢？基本上，你想得到的我都稍微曾經接手過（這也是為什麼我很快就離職跑了）。

以朋友分享來講，如果你是做內容的話，那麼你可能會接觸到以下這幾個動畫技術

  • Cocos Creator
  • Egret
  • Pixi.js
  • Phaser
  • Unity

雖然我主要是負責平台相關的開發，但還是有幫忙協助改一下公司的 Unity 就是了。

Note

這邊朋友也有提到一個有趣的事情，也就是為什麼大多都是 Web 而不是 Mobile APP 呢？ 主要原因是因為如果走 APP 開發的話，必須額外考慮上架的問題，那麼有上架的問題就有可能會被監管，因此走 Web 會比較沒有這個風險。

剛剛前面所聊的主要是客戶端相關的技術，而後端呢？這個就非常看公司，以我先前公司來講是使用 PHP，其他部分就比較看公司團隊的技術，但是有一點是可以比較需要注意的，在博弈產業會比較需要注意瞬時的流量。

那麼後端也很常需要與第三方平台串接，但是依照我自己經驗來講，對接廠商的技術與素質滿有可能踩到雷的（這件事情也剛好朋友有提到），所以在開發上的時間規劃就必須要抓保守一點，否則真的很容易卡在對接上。

接下來講講平台的部分，當時我主要是負責入口網站的開發，雖然有部分會採用前後端分離的方式，但是為了加速開發，所以當時我是採用 PHP + jQuery 的方式開發，這邊也可以稍微提一下，入口網站是博弈產業整個風險最高且收益最高的地方，因為你會需要將公司開發的遊戲上線、引導玩家進入遊戲，又或者是處理玩家的問題等，因此是非常重要的。

金流我就不方便說了，因為這邊如果分享我自己的經驗的話，就真的實在太過於灰色地帶了，我只能說部分博弈產業會為了避免被抓到，會使用各種手段來去閃躲或規避，因此這邊我就不多說了。

所以你說技術面吃得重不重？迭代的速度快不快呢？我自己是覺得滿快的。

博弈產業的待遇

關於待遇部分就是最吸引人的部分了，不外乎你會考慮博弈產業就只是為了一個東西

「錢。」

不可否認博弈產業的薪資會比普通公司來的好，尤其是對於剛轉職、新手來講，可能起薪至少就有相當不錯的數字。

畢竟高報酬就有高風險的問題存在，因此在博弈產業上班我問了三四個朋友，基本上給的答案都差不多，就是...

「可能說抄就抄，平時就要有換工作的心理準備。」

當然每一間公司的福利不同，所以這個真的滿難講的，如果你待的產業是屬於國外博弈的話，基本該有的福利都會有，可能就會少個勞健保之類的，但是在薪水上也不太會跟你凹或砍，跟你談多少就是多少。

那麼前面有提到「平時就要有換工作的心理準備」這件事情，其實我自己在博弈產業待了約半年，離職原因也是突然公司就被抄了，然後就沒了工作，我一整個當下收到通知是滿 WT...

雖然後來當初的博弈主管有想要找我到另一間博弈，但是考慮到每天都要過著提心吊膽的生活，我還是選擇了離開。

那麼這邊還有一件事情要特別提一下，也就是博弈產業的 domain know how 很容易被限制，因此當你離開博弈產業後，你會發現你的薪水會被活生生的砍半，這也是為什麼有些人一直離不開博弈產業了。

Note

這邊要特別提醒一件事情，如果你在博弈產業上班，但是薪水還給得很低的話，就真的完全不推薦你去了，因為你必須要知道你是在用風險換薪水的，如果薪資沒有比一般公司高，那麼你就真的是在白白冒險了。

另外這邊朋友也有稍微提一下他們公司的福利，基本上薪水對於 junior 來講是算滿高的（薪資多少朋友不方便透露，但是以新手來講是真的滿高的。），然後年終至少兩個月以及滿一年有旅遊補助等，當然這一點隨著不同的博弈公司有不同的福利。

（我以前的博弈公司是包中、晚餐都是由公司特別請廚師來做給大家吃的 XD）

「講了那麼多，所以博弈的薪資是多少呢？」

我知道你對於這一點應該是感到非常好奇的，但是這一點我也確實有點難講出來，因為每一間公司給的薪資福利皆不同，身邊的朋友也不方便透露，所以就很難給出一個數字，如果你真的好奇的話，我想你也可以在網路上找到不少相關的資訊。

對新人來講推薦嗎？

為了避免過度主觀，所以這一篇在撰寫之前，我前面有特別提到我有問一下我身邊的朋友，基本上得到的答案是「一半」。

滿有趣的吧？別擔心，這邊我也會分享出為什麼一半推薦一半不推薦的原因。

為什麼推薦

首先先講推薦的原因好了，想進入博弈產業最大的優勢就是薪水，如果你的目標是以錢為優先，並且把錢看得比任何事情還重的話，那麼博弈產業確實是滿推薦的，畢竟就跟投資一樣，風險越高，收益也就越高。

但是請你做好隨時被抄家以及沒工作的心態去看比較好，因為博弈產業的流動率是非常高的，基本上博弈產業都是保持著隨時失去你的心態。

為什麼不推薦

基本上職涯很容易被限制，我朋友分析了三個面向，分別是...

  • 知識
  • 技術
  • 事業

首先我們進入博弈產業後，大部分的知識都會被限制在與博弈相關的領域，而且如果要在博弈長久的話，你必須要建立好知識堡壘的（知道越多，判刑越重）。

技術面基本上也都會遊戲與動畫相關有關，那麼以台灣產業來講，其實是很難找工作的。

最後一點我認為我朋友說的滿對的，你在裡面建立的人脈基本上是不能夠拿出檯面來講的，畢竟樹大招風，所以如果你想建立品牌形象相對是很困難的。

而且離職後，履歷也不好描述自己做了什麼，因為基本上你可能都是在做一些不能夠拿出來講的事情。

進入博弈的風險

那麼前面聊了待遇跟對於新手推不推薦的事情之後，接下來我想你應該很好奇究竟進入博弈有什麼風險呢？

我相信這是大家最好奇的，那麼我問的朋友群中，就真的有一個被判賭博罪的，所以你覺得風險有多大呢？這個風險你能不能接受呢？我認為這是可以留給你自己思考的。

我朋友用自身經驗去分享了自己被判刑的經歷，基本上不管公司怎麼說服你，你都必須要清楚到幾件事情

  • 抄不抄家，檢察官說的算
  • 判不判刑，法官說的算

你必須清楚知道公司不會為你負責，你必須要為自己行為負責任，因為在法律上是有明定知情者並協助賭博者都是屬於有罪的。

所以基本上只要你知情還協助的話，那麼你就無法置身之外，這也是為什麼我朋友會被判刑的原因。

你可能會想說...

「為什麼那麼多人去博弈也都沒事，是不是純粹你朋友衰小？」

我只能說，或許吧？但實際上狀況來講，這個行業本身就是灰色領域，警察要抓不抓都是完全看他們自己，就跟你紅燈右轉警察可能也只是睜一隻眼閉一隻眼，因為這件事情吃力不討好，也不會因此加薪升職，所以完全看他想不想抓而已。

但不可否認一件事情，博弈產界在現實上占了國家一部分 GDP 沒有錯，因此國家是不太可能根絕這個產業的，這件事情我也認同我朋友的說法，畢竟這也跟人性有關。

這時候你也可能會想說...

「為什麼某某某公司在檯面上這麼囂張都沒被抄過家？」

你要思考一件事情，如果那間公司夠龐大的話，你身為警察沒事去抄他們家幹嘛？根本是吃力不討好。

更不用說如果這間公司還不定期贊助政府機關的一些活動、培訓、研討會等等，那麼你覺得警察會去抄他們家嗎？何必沒事抄了自己乾爹呢？你說是吧。

該如何辨別博弈公司？

對於新手或者求職者來講，最困擾的就是怎麼辨別博弈公司，這邊我也分享我跟我朋友的一些辨別方式

  • 公司簡介隨意
    • 正常公司會提供明確的產品跟提供的服務，但是博弈公司通常都是模糊的，因為他們不想被查水表。
  • 公司網站做得很隨性
    • 因為根本不想花太多成本在上面，所以通常都是很隨性的，更不用說有些公司根本就是沒有網站。
  • 充滿著各種福利，但看不到服務什麼
    • 你會看到他的福利超乎想像的好，但是你無法透過簡介找到他們的服務內容。
  • 職缺招募
    • 基本上只要提到「技術客服」至少有 7~8 成就是了。
    • 以前端來講，如果提到 「HTML5/H5(對岸用語)」就有滿高機率是，但如果你看到「Cocos Creator / Egret」 就一定是。

其他辨別方式就是要多打聽了，因為博弈公司通常比較不會在檯面上講，所以你可以加入一些社群去打聽消息跟風聲，畢竟網路上也是有人會去分享有哪些公司是博弈公司。

當然你也有可能遇到一種公司，他會跟你說他是接案公司，但實際上進去後你發現你做的是跟博弈相關的，但是在這邊我也不想一竿子打翻一船人，因為有可能這間接案公司就真的只是接案公司，只是他的客戶是博弈公司，又或者他的客戶群比較多元，只是剛好博弈產業佔了大多數而已。

所以以我自己認知中來講，如果你真的完全不想碰到博弈相關，又或者你無法接受的話，那麼在面試的時候就要多問一些問題，像是

  • 公司的客戶群有哪些？
  • 公司的主要產品是什麼？
  • 公司的主要服務是什麼？
  • 這個職缺主要會負責什麼？

透過詢問的方式盡可能問出來，畢竟你也不想到時候進去後才發現，那時候你就只能自求多福了。

當然你也可以乾脆考慮直接切割，打從一開始就表明自己不想接觸與博弈相關的開發，如果真的好死不死的進去了，那麼你也可以考慮離職。

請把握一個原則：

公司可以選擇你，你也可以選擇公司。

千萬不要硬著頭皮硬做，讓自己每天活在驚心膽跳之中，只要你打從一開始心裡就無法接受了，那麼就不要去了，就這麼簡單。

雖然你可能已經盡可能去打聽風聲跟尋找相關資訊，但還是不小心進入或接觸到的博弈產業的開發，那這種狀況下原因是為什麼呢？因為比較大且有規模的博弈都是將功能分得比較細，因此你也不一定會了解或接觸到整個業務全貌，你可能接觸到的部分只是一小部分而已，畢竟要做好風險管控，所以如果真的要去的話還是會建議找到大型的博弈公司會較好，畢竟人家能夠做到那麼大了，黑白兩道加減都有認識。

但這也是好處，做最少的事情就可以領很多的錢，但缺點就是很難有機會學習到東西。

Note

這邊朋友也有特別提到一件事情，也有一種接案公司是專門負責博弈公司的周邊事業，例如後台操作系統、網路安全管理等等，但這個就不提了，因為水太深了。

結論

那麼為什麼我會分享這一篇呢？其實身邊人一直在問一些問題，例如...

  • 進去博弈會不會抄家？
  • 會不會被抓？
  • 到底有沒有犯法 ...

諸如此類的，所以就乾脆寫一篇分享出來，畢竟這也是一個產業，只是這個產業比較灰色而已，但是這個產業也是有他的存在價值，所以我也不會去評論他們的對錯，只是單純分享一些我知道的事情而已。

最終決定是否要進去做的人是你，我希望你可以透過這一篇了解到預期可能會遇到哪些風險跟問題，如果只是為了還債、高薪的話，那麼也應該要做好哪些心理準備，基本上我都會跟我的學生小聊一下這一塊，我自己是希望真的想進去博弈產業的學生，可以有一個心理準備，畢竟這個產業是有一定風險存在的。

最後的最後，我還是要強調一件事情

「我並沒有支持正在閱讀這一篇的你去博弈產業，也沒有反對你去博弈產業，而是希望透過這一篇文章讓你知道進去這個產業可能會有哪些風險，如果有一份正常且高薪的工作，那麼有誰願意犯險呢？」
