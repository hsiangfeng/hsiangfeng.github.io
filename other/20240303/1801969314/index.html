前言

家裡其實有兩台 Windows 系統的電腦，雖然還沒上 Windows 11，但 Windows 的強制更新機制真的很令人頭痛，因為 Windows 的強制更新機制會導致我們電腦用到一半變得很慢，或者更新之後出現一推怪問題，所以這一篇就來記錄一下如何關閉 Windows 10 的強制更新機制。

提醒事項

這邊還是要提醒一下，關閉 Windows 10 的強制更新機制會導致你的電腦可能會有安全性的問題，所以如果你不是很確定自己在做什麼的話，建議不要隨便關閉強制更新機制會比較好。

（謎之聲：但 Windows 更新失敗機率超級無敵高啊！～～～～～～～）

關閉更新機制

首先我們要先了解一下為什麼要有關閉更新機制這個需求，不外出原因有這幾點：

  • 電腦變慢
  • 網路頻寬佔用
  • 電力耗損增加

而且更新過程也非常的耗時，如果剛好是緊急狀況需要馬上用電腦時，你就只能看著你的電腦更新了。

最慘還是更新錯誤/失敗的問題，這時候你就會看到令人討厭的「藍屏」畫面

Note
藍屏是指 Windows 系統發生錯誤時，會顯示一個藍底白字的畫面，上面會有一些錯誤訊息告知你發生了什麼問題，但是對於沒有相關知識的人來說，這個畫面就是一個很大的問題，因為根本不知道該如何解決。

但這邊還是要再提醒一下，如果你關閉了 Windows 10 的強制更新機制，你的電腦可能會有安全性的問題，或者發生某些軟體無法使用的問題，所以如果你不是很確定自己在做什麼的話，建議不要隨便關閉強制更新機制會比較好。

如果沒有任何問題的話，就讓我們開始關閉 Windows 10 的強制更新機制吧！

第一步驟：開啟「服務」視窗

請在你的 Windows 10 電腦上按下 Win + R 鍵打開「執行」視窗，然後輸入 services.msc 並按下 Enter 鍵

如果找不到 Win 鍵的話，你可以再開始工具列上搜尋「執行」這樣也可以開啟「執行」視窗

當你按下 Enter 鍵之後，你會看到一個「服務」的視窗，這個視窗就是 Windows 10 的服務管理視窗

第二步驟：找到「Windows Update」服務

在「服務」的視窗中，你會看到一個「Windows Update」的服務，這個就是 Windows 10 的更新服務，如果不好找的話，你可以在這個視窗按下 W 鍵，然後你就會看到所有以 W 開頭的服務

你可以看到它目前的狀態是「執行中」與「啟動類型」是「自動」，這就是 Windows 10 的強制更新機制

第三步驟：關閉「Windows Update」服務

接下來請你右鍵點擊「Windows Update」服務，將它的「啟動類型」改成「手動」 or 「停用」，並且將「服務狀態」改成「禁止」

這樣你就完成了關閉 Windows 10 的強制更新機制哩！

Note
雖然我們已經設定了「Windows Update」服務的「啟動類型」為「手動」 or 「停用」，並將「服務狀態」改成「禁止」，但是你只要進入到「Windows Update」的設定畫面，它就會自動啟動唷。
