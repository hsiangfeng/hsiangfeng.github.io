Discord 是什麼呢？

Discord 本身原本定位是遊戲玩家的語音軟體，但是隨著時間推移，Discord 也推出了許多很好用的功能，讓 Discord 逐漸成為一個社群軟體，但是這也代表著 Discord 會有許多問題，這邊讓我來稍微整理一下常見的問題。

當然，如果你不熟 Discord 的話，也推薦你閱讀以下文章

  • Discord 好難？不怕！我教你！
  • 快速入門 Discord 社群通訊軟體

Discord 無法登入

Discord 帳號無法登入的狀況基本上不外乎有以下兩種

  • 帳號被鎖定
  • 密碼忘記

如果是單純的密碼忘記，其實就只需要重設密碼而已，但是如果是帳號被鎖定，那問題就不一樣了，因為通常帳號鎖定的原因大多是違反了 Discord 的規章，最常見就是使用機器人大量發送訊息，或是使用機器人大量加入伺服器，這些都是 Discord 不允許的行為，所以如果帳號被鎖定，那就只能寫信給 Discord 官方，希望他們能夠解鎖你的帳號。

無法接受邀請

這種狀況最常見於點了伺服器邀請連結卻無法加入的狀況，而這種狀況有以下幾種可能

  • 被該伺服器封鎖
  • 該伺服器有設定限制

如果是被伺服器封鎖的話…你就只能私訊或想辦法聯絡上該伺服器的管理員，但是基本上會被該伺服器封鎖不外出就是做出了一些違反他們伺服器規則的行為，通常很常見就是帳號被盜才被封鎖，如果沒有違反他們規則，也沒有被盜用的話…那恭喜你，你只是純粹被討厭了而已。

如果是伺服器限制導致的話，那麼你可以先釐清一下該伺服器有哪些規則限制，基本上 Discord 提供以下驗證的等級

分別是…

  • 低 => 必須驗證 Email
  • 中 => 必須驗證 Email + Discord 註冊時間要超過五分鐘
  • 高 => 必須驗證 Email + Discord 註冊時間要超過五分鐘 + 成為伺服器成員要超過十分鐘
  • 最高 => 必須驗證 Email + Discord 註冊時間要超過五分鐘 + 成為伺服器成員要超過十分鐘 + 手機驗證

通常來講至少會開到「低」這個規則，如果你是一個新註冊的帳號，基本上你應該會在畫面上看到 Discord 一直提示你要驗證 Email，如果你是一個舊帳號，那麼你就要確認一下你的 Email 是否有被驗證過，如果沒有的話，那麼你就要先驗證 Email 才能夠加入該伺服器。

除此之外，這個驗證規則也會影響你是否可以在該伺服器發送訊息唷～

Note
這個功能主要是為了避免廣告機器人，如果為了方便使用者而取消的話，反而會讓這個社群健康度大幅度降低，因此大多 Discord 社群都會開啟這個功能。

該如何離開語音頻道

有時候我們在看伺服器的時候，會不小心進入到語音頻道，這時候你會發現你的左下角會出現這個圖示

那該如何離開這個頻道呢？其實很簡單，點一下掛斷電話的符號就可以中斷連接了

也題外話一下，如果你只是想要旁聽而不想要說話的話，你可以點一下下方的麥克風圖示，這樣你就不會被其他人聽到你的聲音囉

（另一個就是關閉收音，但我認為關閉收音不如就離開頻道吧）

語音沒聲音/聽不到聲音/出現雜音

如果你是一個很常使用 Discord 聊天的人，基本上就會很常遇到這個問題，通常很常發生在你已經進入語音頻道，然後接上麥克風/耳機導致 Discord 無法正常辨識你的麥克風/耳機，接著就會發生你無法聽到別人說話或別人無法聽到你說話的狀況。

解決方式很簡單，當出現沒聲音/聽不到聲音/出現雜音的時候，你只需要打開 Discord 的設定

找到「語音 & 視訊」這個選項

接著將你的輸入/輸出裝置切換一下就沒問題了

透過這個方式就可以讓 Discord 重新辨識你的麥克風/耳機，如果還是不行的話，那麼你就要檢查一下你的麥克風/耳機是否有問題囉。

無法私訊特定人士

基本上這問題是出在於該使用者的「隱私 & 安全」有調整過導致的，在使用者設定中，你可以找到「隱私 & 安全」這個選項，底下有一個「允許來自伺服器的私人訊息」

如果你是關閉的話，基本上是必須先加你好友才能夠私訊你，如果你是開啟的話，那麼你就可以直接私訊該使用者了。

陌生訊息不見了

Discord 後來改版多一個功能叫做「訊息請求」，這個訊息請求主要是針對陌生訊息，如果你不是朋友的話，那麼你的訊息就會被歸類到「訊息請求」中

這個功能類似於 Facebook 的「陌生訊息」，基本上你只要回覆他或者按下 Accept DM 就可以在訊息列表上看到他的訊息，主要是為了減少廣告機器人的問題。

但如果你不要這個功能的話，你可以在使用者設定中，找到「隱私 & 安全」這個選項，底下有一個「針對你可能不認識的伺服器成員，啟用訊息請求」，你只需要關閉這個功能就可以解決陌生訊息都被歸類到「訊息請求」的問題了

文字、表情符號消失

基本上如果是表情符號、文字或各種奇怪看不到的問題的話，大多這時候你只需要重新整理 Discord 就可以解決了，快捷鍵分別是以下

  • Windows => Ctrl + R
  • Mac => Command + R
