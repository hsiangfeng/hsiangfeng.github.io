前言

截圖是一個很常使用的功能，所以就寫一篇分享一下 Mac、Window 的電腦截圖，可以的話也請盡可能避免使用手機拍照提供畫面唷~

macOS 截圖

Mac 本身就有內建非常好用的截圖與錄影功能，只需要按下組合鍵即可開始，底下我也先列上組合按鈕

  • Command + Shift + 3
    • 擷取當前整個視窗
  • Command + Shift + 4
    • 擷取螢幕的一部分
  • Command + Shift + 5
    • 畫面錄影與擷取工具
  • Command + Shift + 6
    • 擷取觸控列

而我比較常用的主要是「擷取螢幕的一部分」與「畫面錄影與擷取工具」，而這兩個組合按鍵也是我推薦你必須一定要學會的。

這邊要注意使用 Mac 內建截圖的副檔名是 .png 而錄影檔案為 .mov，檔案名稱通常包含當下的日期和時間。

擷取螢幕的一部分

最簡單的截圖方式就是按下鍵盤上的 Command + Shift + 4 (選取截圖) 組合按鍵，此時你就會出現一個選取視窗，這時候選取你要擷取的部分就會自動擷取並儲存檔案囉~

但是如果你不想要儲存而是單純的將當前選取的畫面擷取下來，那你可以 Command + Shift + 4 組合按鍵後出現選取鼠標時再補按著（請不要放開） Control 再去選取要擷取的畫面，此時就可以將圖片直接複製到剪貼簿（記憶體）內而不會儲存檔案囉~

我知道這有點難度，所以我附上 Gif 給予參考

畫面錄影與擷取工具

當你按下 Command + Shift + 5 組合按鍵 (畫面錄影與擷取工具)後，就會看到畫面上出現這一排

由左邊到右邊的功能是以下

  • 擷取整個螢幕
  • 擷取視窗
  • 擷取螢幕的一部分
  • 錄製整個螢幕
  • 錄製螢幕的一部分

其他部分如果你在錄製畫面時，也希望錄製聲音話，可以點選選項去做調整。

如果你覺得 Mac 內建截圖真的很爛的話，你可以考慮安裝「Skitch」

Microsoft Windows 截圖

Windows 每一代都有內建截圖功能，也就是鍵盤上的 Print Screen (也可能標示為 PrtScn 或 PrtScrn) 鍵，另一種則是使用內建的「剪取工具」，或者使用 [Windows 鍵]+ Shift + S 開啟擷取工具也是一種方式。

Windows 10 之後本身內建是有錄影的工具，只需要按下 [Windows 鍵]+ Shift + G 就可以開始開啟錄影工具囉。

如果你嫌內建太爛的話也可以考慮付費購買 SnagIt(後來也有支援 Mac OS)。

LINE 截圖工具

如果你真的真的不想安裝任何軟體，但又覺得內建很爛的話，我推薦你使用 LINE 截圖工具，打開任何一個聊天視窗，下方第三個就是截圖工具

不論你使用哪一種截圖方式都遠比你使用手機拍來的好太多，而且也不用擔心螢幕反射不小心露出背景或是肥宅真面目。

所以我認真的說「能使用截圖工具就請使用截圖工具」。
