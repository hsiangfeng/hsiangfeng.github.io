前言

不論是在 Mac 或是 Windows 開發，我們一定很常使用終端機(Terminal)或是命令提示字元，而往往畫面就會非常多資訊看了很阿雜，所以紀錄一下有哪些方式可以清除終端機畫面。

清空終端機

首先我們終端機畫面上可能一大推資訊

雖然不影響操作跟使用，但是看多了就是覺得煩，你可以試著輸入以下指令來清空畫面

  • Mac
    • clear
  • Windows
    • clear
    • cls

Note
透過輸入指令的方式並不會真的清除終端機的內容，只是將畫面往上移動，所以如果你往上滑的話，還是可以看到之前的內容。

但是這邊其實還有另一招方式可以快速清除終端機的內容，也就是按下組合鍵

  • Mac
    • Command + K
    • Control + L
  • Windows
    • Ctrl + L (早期似乎是 Ctrl+ K)

以上是清除終端機內容的簡單方式。
