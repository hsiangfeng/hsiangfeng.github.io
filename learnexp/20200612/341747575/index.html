入職後發現技術不足怎麼辦？

發表於 2020-06-12 更新於 2020-06-13 分類於 學習經驗

這幾天有幾位類似相同境遇的學員來跟我小聊一下自己工作近況，話題大多都是關於「入職後發現自己技術完全跟不上，同事開會時也幫不上忙，是不是該離職了。」

前言

這幾天有幾位類似相同境遇的學員來跟我小聊一下自己工作近況，話題大多都是關於「入職後發現自己技術完全跟不上，同事開會時也幫不上忙，是不是該離職了。」

入職

通常「入職後發現自己技術完全跟不上，同事開會時也幫不上忙，是不是該離職了。」這種狀況其實很容易發生在自己的第一份工作，因為我們常常把工作以及職業想像的太美好，例如工程師就是高薪、快速打鍵盤畫面瘋狂跳 Code 出來好像很屌一樣，但現實往往是不同的。

我還記得自己當初剛退伍時的第一份工作是寫 C#，老實講我早期第一個程式語言是學 PHP，因此當初面試第一份工作時，並沒有特別說工作內容會使用其他語言並且面試的上機考也是用 PHP 寫，所以我就認為應該開發上也是用 PHP 吧？

後來當然也很幸運的錄取了，可是殊不知錄取後才是悲劇的開始。

報到的第一天我立刻被通知要改寫另一個程式語言也就是 C#，在轉當前端工程師之前其實我完完全全沒有接觸過 PHP 之外的第二語言，因為剛退伍很菜(我承認)，雖然我後來陸續學了 Python、C 等(但都只是皮毛畢竟沒有專精)。

就職過程中其實碰壁的非常非常的嚴重，因為自己完全沒有摸過這個語言。

雖然公司有請一位前輩帶我，但前輩本身也忙於自己的專案開發，所以其實也沒什麼空理我(一個禮拜大概說不到兩次話吧？)，因此上班的時候有八個小時我都在拼命想辦法學習 C# 以及 MVC 到底是什麼鬼東西。

這份工作雖然僅任職了半年，但這過程我心中曾經萌生出過很多負面想法，例如：「我真的適合當工程師嗎？」、「我是不是該放棄這一條路」以及「是不是試用期到後自己提離職比較好」等負面想法。

焦慮

第一份工作帶給我很嚴重的焦慮問題，焦慮的程度其實有讓我情緒幾度差點崩潰，因為公司本來打算讓我這個位置的人接手一個大型關案，但因為我沒有相關 C# 開發經驗，所以這個專案的開發都交由給了另一位同事。

過程中其實我一直在瘋狂 Google 該如何學習 C#，但說真的一個月內要學會一個自己完全不懂的東西真的有點困難，更不用說我連開發架構 MVC 是什麼都不懂，而且當時線上教育資源真的很少，通常只能夠透過書本來學習，還不用說自己菜到不行(當時我連 Git 都不會)。

在時間的壓力以及技術不足的狀況下，到職後約一個月，迎來了第一次的開會，我深刻體會到自己的無力，因為自己完全聽不懂同事到底在講什麼，自己只能默默的坐在旁邊像個可達鴨抱著頭，其實這時候我已經焦慮到一個極致，因為我看到了幫我頂專案的同事做到快崩潰的樣子我心裡更是覺得自己很廢物。

直到我任職第二個月的時候，我鼓起勇氣面對自己的焦慮以及調整自己心裡的狀況後開始有了一點改善。

改善

我記得我當時因為已經不知道方向也不知道該如何學習，當時我連 Google 要搜尋什麼關鍵字都不知道，可是又不想放棄這一份工作，所以我鼓起勇氣使用公司內部通訊軟體敲了前輩

「前輩，不好意思打擾你了，我想跟你請教一下該怎麼學 C#，不知道能不能請你指引我一些方向？」

其實這一句話我當時猶豫了半小時才送出，因為大家都在負責大型專案，公司氣氛非常凝重，所以我一直不敢打擾他們，當我送出訊息之後等了約 10 分鐘左右一直沒看到前輩的回信，心裡正要絕望時，前輩回了我訊息

「你先不用管 C# 怎麼學，你先搞懂 MVC 再說…」

因為我的鼓起勇氣送出的一句話，前輩開始指引我可以看哪些書以及資源，而過程我的焦慮也開始慢慢減少，因為我的焦慮就是來自於自己的技術能量不足所導致，所以我其實一直都不肯面對自己的焦慮原因在於「我害怕影響到別人」、「我不肯面對自己很廢的事實」，所以才會導致出各種負面能量的產生。

轉變

因為開始面對自己的焦慮之後，也肯定「自己就是爛死」的想法，而讓我開始有了很大的轉變，我開始不要命的瘋狂和前輩請教他各種東西，其中包含了伺服器架設、GCP 操作以及開始將公司專案從其他語言改成 C#.MVC。

雖然這一份工作僅僅只待了半年，但其實收穫真的很大，我也開始慢慢的可以跟前輩以及其他同事討論一些程式相關的問題（雖然還是很菜），但是與當初剛入職時的感受完全已經不一樣。

過程中我也體會到一件事情，有時候我們會無形中產生各種負面想法以及焦慮，但是如果一直不去面對自己焦慮問題的原因，其實只會徒增自己的負面能量進而導致放棄的念頭，所以我當時停下腳步先從自己的作息開始調整，確實有一種莫名其妙的無腦勇氣詢問前輩(現在想想當初大家忙得要死，卻一直打擾他們真的感到很抱歉)。

也因為我承認了自己真的很廢很爛，所以也抱著前輩會開罵的心態去敲他，雖然前輩沒有開罵我就是了，可是我心裡的焦慮也開始減少。

過程

這一份工作中，我認知的體會到一件事情，當你發現自己好像越來越多負面能量時，你必須立刻停下腳步讓自己休息一下，往往我們都不太會注意到自己已經深陷在負面能量的場景中，只要發現自己已經開始因為某件事情開始睡不好、吃不好以及心裡好像壓著一個石頭時，其實你就真的該休息一下思考自己焦慮的原因。

有時候我們必須給自己一個適當的壓力才能夠成長，但過度的壓力反而會得到負成長，我們必須認知自己焦慮的問題來源是什麼，是不是技術不足？是不是自己給自己壓力太大？等，當我開始面對焦慮時，我先是回顧思考了一下自己是為了什麼而來這間公司以及這間公司的錄取機會是我自己努力爭取來的，為什麼我要繼續這樣子放任自己焦慮而不去正視它呢？

過程中一定有相當多人會看不起，但也因為有人看不起才更要去突破，因為當你突破時那些看不起你的人就會反過來崇拜你。

此外當一間公司願意錄取你時，就代表公司其實願意花時間培養你信任你，公司絕對是看到你身上某一個特質而錄取你，就算僅僅只待了試用期三個月，但這試用期如果你好好把握卻也能夠改變你接下來的一生，因為當你自己親眼看到真實的狀況時，你才會有方向去思考自己該往哪裡精進。

最後真心的希望這一篇文章能夠給此時此刻看得你一點點鼓勵也好，因為我想讓你知道你絕對不是孤單的，因為我也是一樣，所以我更希望你不要因為自己一時的負面想法而導致自己後來的後悔以及遺憾。
