oh！不，電腦中毒了

我國中到高中那段時間，USB 幾乎是傳檔神器，但因為要一直在不同電腦之間插來插去，它同時也成了最強的 『煉蠱罐』 。特別是在網咖，電腦裡的病毒種類多到像是開盲盒，你永遠不知道下一次 USB 插進去會中什麼獎。雖然網咖老闆也會想辦法防堵，有的用膠帶封住 USB 插槽，有的乾脆拔掉前方插孔，但這些小伎倆對後面的 USB 插槽完全沒輒，該中毒還是照樣中毒。

畢竟很多裝置都是透過 USB 連接的，例如鍵盤、滑鼠，所以有些網咖乾脆把整台主機鎖進櫃子裡，然後把鍵盤、滑鼠的線拉到櫃子外。這樣一來，玩家就沒辦法隨便插 USB 了。

不過我常去的那間網咖可沒把主機關起來，整台就大剌剌地曝露在外面。於是我們每次都自己帶 USB 去插，然後再把這顆插過網咖的 USB 帶回家，甚至拿去插學校電腦。結果 USB 根本就變成了 『病毒快遞服務』 ，專門幫病毒從網咖送到家裡，再順路送到學校。

「學校電腦不會擋嗎？」你可能會這樣問。

老實講呢…不會。

其實並不是學校電腦管制不嚴，而是因為課堂作業幾乎都要靠 USB 來繳交，所以電腦一定得開放 USB 插槽，方便學生把作業存進去再交到學校電腦。結果也因此，學校電腦常常跟網咖電腦得同樣的病症。而說到這件事，就讓我想起以前的一個小故事。

「ㄟㄟ！XXX 你有沒有帶魔獸啊？」電腦課上，我問了一下坐在對面的同學。

「有啊，在這裡，你要打哦？」同學一邊拿給我，一邊回答我。

「對啊，來打啦！」接過同學的 USB 後，我馬上就插到電腦上，然後 Copy 到學校電腦上開打魔獸爭霸。

對了，早期我們電腦教室的座位都是固定的，所以誰的電腦出事都可以知道是誰的問題。

我記得當天打完遊戲之後，要打開瀏覽器時，就發現不太對勁….

因為我的網頁變成了：

通常學校瀏覽器會有兩種首頁

  • 學校官方網站
  • Yahoo！ 首頁

但我打開卻變成了「hao123.com」，當時我並沒有想太多，我就把它關掉，然後重新設定首頁後繼續上課。

只是後來發現越來越不對勁，因為我的電腦開始變得很慢，然後瀏覽器打開又是「hao123.com」，這時候我才意識一件事情。

「Ｘ！我的電腦中毒了啦！」我跟對面同學說。

「你也中毒了？！」同學一臉驚訝的看著我。

不出意外，果然還是出了意外。我們完全沒注意到老師已經走到身邊，結果他一看到我們兩個電腦的瀏覽器首頁都是 hao123，立刻要我們換電腦，還馬上通報資訊中心來處理。

那為什麼會特地提到這件事呢？因為在某一天，我爸又帶了一台新電腦回家，而我偏偏又把那支中毒的 USB 插了上去。結果可想而知，這台電腦也跟著中毒了。

也因為電腦中毒導致越來越慢以及被綁架首頁，所以我就被迫學會了怎麼重灌電腦。

電腦病毒

簡單來講，電腦病毒就是一個惡意的程式碼/軟體，它的概念跟感冒病毒一樣，都需要一個宿主來寄生，然後再透過這個宿主來傳播到其他的電腦上。

而電腦病毒最主要目的是破壞系統、竊取資料、或是讓電腦變得無法使用，這些都是電腦病毒的常見行為。

「有一段時間最嚴重的電腦病毒，大概就是勒索病毒了。它會把你的檔案整個加密，然後要求支付贖金才能解鎖。這種病毒特別可怕，因為一旦中招，你的檔案就會被鎖死，完全無法使用。」

感染途徑

  • 檔案下載： 在執行檔（.exe）or 安裝檔（.msi）裡，有些網站甚至會誘導你下載看似安全的檔案，其實暗藏病毒。
  • Email： 病毒常透過附件傳播，偽裝成 Word、Excel 或 PDF 文件，一旦打開就可能中毒。
  • USB 隨身碟： 在學校或網咖最常見，通常會透過「autorun.inf」自動執行，讓 USB 成為病毒的最佳媒介。
  • 網頁瀏覽： 早期的 Flash 與 JavaScript 漏洞很容易被利用，病毒就能藉此進入電腦，這也是現在瀏覽器全面淘汰 Flash 的原因。
  • 軟體漏洞： 系統或軟體版本過舊時，駭客會利用漏洞入侵，因此定期更新非常重要。
  • 社交工程： 攻擊者假冒朋友或同事傳送檔案，引誘你打開，藉此散播病毒。
  • 網路分享： P2P 軟體（如 BT、eMule）下載檔案時，常常會順便把病毒帶進電腦。
  • 公共 Wi-Fi： 這個就比較特別一點了，駭客會設置假熱點進行「中間人攻擊」，竊取資料或植入病毒。

造成的影響

  • 效能影響： 會佔用 CPU、RAM、硬碟資源，讓電腦變慢甚至藍白畫面（BSoD）。
  • 資料損失： 會刪除檔案，或像勒索病毒一樣把檔案鎖住。
  • 隱私洩漏： 會竊取帳號、密碼、信用卡號等敏感資訊，用於詐騙或盜刷。
  • 網路安全： 把電腦變成「肉雞」（殭屍電腦）又稱魁儡，被駭客用來發動 DDoS 或寄垃圾信。(這些行為會讓你的電腦變成駭客的工具，然後去攻擊其他的電腦或伺服器)
  • 硬體損壞： 過度消耗資源造成過熱，嚴重時可能損壞主機板、顯示卡。(情況少見)

病毒類型

  • 特洛伊木馬：偽裝成正常軟體，引誘使用者執行。
  • 蠕蟲病毒：可自我複製，快速在網路中傳播。
  • 勒索病毒：加密檔案並索取贖金。
  • 廣告軟體：強制彈出廣告。
  • 間諜軟體：竊取使用者隱私資料。
  • Rootkit：隱藏自身，取得系統最高權限。

預防方式

  • 定期更新系統與軟體： 這是最基本的防範措施，許多病毒會利用系統漏洞入侵，定期更新能有效降低風險。
  • 安裝防毒軟體： 防毒軟體能即時偵測並清除病毒，提供基本防護。
  • 不隨便下載檔案/未授權網站： 利用的是使用者貪小便宜的心理，下載破解檔、免費軟體常藏病毒，下載來源一定要謹慎。
  • 啟用防火牆： 防火牆可監控網路流量，阻擋不明連線。
  • 不隨便插 USB： 不要亂插來路不明的隨身碟，以免帶毒。

但不管怎麼防，肯定還是會有一些意外發生，所以請務必做好定時的備份工作，這樣才能避免資料損失的風險。

Note
備份方式有分成本地備份、異地備份、雲端備份等，使用者可以根據自己的需求來選擇適合的備份方式。

重灌系統

重灌系統，說穿了就是「把電腦砍掉重練」。

但這邊所謂的砍掉重練並不是叫你拿刀子去砍電腦…

而是將電腦中的所有資料與作業系統全部刪除，然後重新安裝作業系統與軟體，簡單來講就是恢復「出廠設定」的意思，如果稍微有點難懂，你也可以理解成「格式化硬碟」的概念。

那麼我們該怎麼重灌系統呢？在開始重灌之前需要先做以下事情：

  1. 備份重要資料： 這是最重要的一步，因為重灌會清除電腦裡所有資料，先把重要文件、照片等存起來。
  2. 準備安裝媒介： 準備一個可以用來安裝作業系統的媒介，例如 USB 隨身碟或 DVD 光碟，並把作業系統的映像檔（ISO）放進去。
  3. 取得作業系統： 以 Windows 為例，可以到 官方網站 下載 ISO，然後製作成 USB 開機碟或燒錄到 DVD 光碟。
  4. 設定 BIOS： 不論是選擇 USB 隨身碟或 DVD 光碟作為安裝媒介，進入 BIOS，將 USB 或 DVD 設為第一開機裝置，常見的進入方式是在開機時按 F2、Del 或 Esc。
  5. 開始安裝： 當你完成以上步驟後，就可以開始安裝作業系統了，這邊以 Windows 為例，從安裝媒介啟動後，會進入安裝畫面，按照指示一步步完成作業系統安裝即可。
  6. 安裝驅動程式： 早期的作業系統會需要額外自行手動安裝驅動程式，但現代作業系統大多自帶常用驅動程式，可以正常開機使用；若有少數硬體沒驅動，則再手動下載安裝即可。

以上就是重灌作業系統的流程，但其實這個流程會因為作業系統的不同而有所差異，例如：Linux、macOS 等等，這邊就不一一贅述了。

而重灌系統的好處有很多，例如：

  • 清除病毒： 重灌能有效清除大多數病毒，讓系統恢復正常。但若是勒索病毒，檔案已被加密，就算重灌也救不回來。
  • 提升效能： Windows 用久了會因安裝/移除軟體殘留檔案而變慢，重灌能讓系統回到初始狀態，效能自然提升。
  • 修復系統問題： 藍屏、無法開機等多因系統檔案損壞，重灌大多都能直接解決。
  • 更新系統： 重灌通常會順便升級到最新版本，享有新功能與安全性更新。

結語

不知道大家有沒有重灌電腦的經驗？

我大概平均兩年就會重灌一次電腦，主要原因是 Windows 用久了很容易累積各種垃圾檔，而這些檔案要清除起來特別麻煩，所以乾脆定期重灌，讓電腦保持在比較乾淨的狀態。

不知道有沒有人跟我一樣呢？
