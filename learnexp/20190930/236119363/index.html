被車輪戰的六角同學會

#其他 #學習經驗

前言

在 107/09/29 教師節當天六角學院舉辦了六角同學會，過程有一個自由聊天時間，所以這一篇就來記錄一下我被學員們車輪戰的過程吧(?!)

六角同學會

前面稍微廢話看一下六角學院的同學會吧~

六角學院在教師節當天舉辦了同學會，一開始我還在擔心會不會因為天氣變化太大而導致學員的報到率會比較低，實際上其實報到率還是滿高的呢!

人數大概有 1XX 位左右，實際我也忘了多少人，只是照片後來我就沒有補充了~

同學會過程還邀請了先前的五位學長姐們回來分享自己的就業、面試 & 學習心得給在場得學員們，內容真 D 精采~

最後不免當然有一些點心，可惜我沒有拍很多，所以就附上兩張食物照片聞一下吧

同學車輪戰

接下來在後面主題有一個同學聚會聊天，這也是這一次同學會最主要的舉辦目的，你可以線下面對面找你一直想要聊天得老師、助教 or 學長姐們聊聊，請他們分享能量給你們，一開始我其實是打算躲在角落當邊緣人然後耍廢滑手機 (Ya)

可是實際上我錯了，我立刻被 包圍車輪戰 Play !!

當下我分享的時候可能會因為一時的金魚腦而導致自己不知道該怎麼解釋甚至口誤，所以我決定整理一下學員問我的部分以及我給予的建議，希望可以分享給更多想來卻沒有來但是想問我類似問題的同學們。

車輪戰 START

由於詢問的人來來去去也有點不記得是誰了，所以就不列出提問得學生了，而且有可能會少一些提問，因為有點多(金魚腦無誤)。

學習上卡關

學員：我現在在看 JavaScript 學徒試煉，小作業幾乎都做得出來，但是如果要做一個大作品卻會出現卡關甚至做不出來。

我：這個問題最主要是出在你 練習不夠 (講幹話)，其實這種狀況我在早期學習 PHP 的時候也有過這種問題，當時我想撰寫一個作品時腦袋就會卡關，後來我發現最主要的問題是出在 我沒有將功能拆出來做，那這是什麼意思呢?首先原本你可能要開發一個功能，而該功能分別為 A、B、C、D 四大功能，但是你在思考的時候可能是 一次在想這四個功能怎麼做，但是其實這就是最主要問題，你應該將功能一個一個切離出來，一次只思考一個功能該如何製作，而不是大方向得去思考這四個功能該怎麼做，所以正確的方式應該是去思考 A 如何製作，製作完再去思考 B 以此類推，這樣子你也比較知道該如何製作且也比較不會出現自信心受創的問題。

目標達不到

學員：我發現我訂了一個目標，但是我無法達成我想請問一下一般這種狀況你會建議調整目標，還是去不調整目標蒐集資料後再去繼續原有目標?

我：我想先問一下在場各位有哪幾位學員已經開始學習 Vue 或是有看過我得 Ro 仙境傳說電商呢?

(舉手)

我：竟然絕大部分人都有看過(覺得驚訝)，其實我一開始在製作 Vue 作業的時候目標是使用 Canvas 來製作一個 RO 電商，可是我發現我根本不會 Canvas 而且要以現階段有限的時間內學習 Canvas 並製作出一個作品更是不可能的任務，所以我就決定稍微調整一下目標改用我會得技能折衷製作，並且盡可能得還原遊戲中的畫面，也因為這樣子我的作品在 Vue 與 Vue2 得畫面呈現上就會有非常大的差異，但是你可以發現我會離我的目標距離更近一些，所以我個人會建議當發現目標達不到的時候你可以嘗試稍微調整一下目標，當然你想要保留既有的目標也是可以的，只是通常來講我個人不建議這麼做，因為當你發現目標一直無法達到你會越來越沒有信心以及挫折感，簡單來講許多事情你不能去要求十全十美，但是你可以要求自己好上更好，我的作品就是一個範例可以給你參考，當然這只是我的一個建議，你可以依照自身狀況去做調整。

該如何與大神建立連結

學員：我都會參加一些研討會以及聚會，但是我總是無法與這些大神有所講話，想請問該如何與這些大神建立連結?

我：其實很簡單，認識這些大神的方法非常多，你可以直接去他 FB 按下送出好友(超認真講幹話)。

(全場笑翻)

我：如果你真的想要與這些大神建立一個連結 or 弱連結那就試著去 犯錯，當然不是叫你蓄意犯錯，而是不要害怕犯錯，你可以試著去參加一些活動，舉例來講，我這一次鐵人賽報名了兩個主題，一個主題是後端主題，另一個則是 JavaScript 前端極新手的主題，雖然我已經懂一些核心觀念，但是其實我文章中還是會有寫錯的地方，而這些地方也引來了兩位大神的指教，這過程你也會因為這些大神的指教而成長

我：其實只要你願意公開來嘗試撰寫，並勇於接受自己的錯誤，你就有很高的機會與這些大神接觸，當然還有另一種方式也就是試著去看這些大神曾經分享過什麼，你可以再他分享的文章下面提出自己的看法，舉例來講前陣子胡立有分享 異世界?r3:0挑戰 這個遊戲，我覺得非常有趣，所以就去私下密一下胡立希望他可以允許我將破關過程寫成文章，過程也為了尊重胡立，所以就會告知他文章撰寫完畢時，也會給他看過。

(關於我逃出胡立的異世界 R30 這件事之攻略包)

我：這過程你就已經與這些大神建立了一個弱連結，所以與這些大神接觸的方式有百百種，當然你也可以多參加六角學院的 大神來六角(置入行銷)，你如果常常在上面發言，講師自然也會對你有點印象，就像你們對我有印象一樣，而這也是與講師建立連結的一種方式。

薪資該怎麼談

學員：我不知道該怎麼與面試官談薪水才妥當?

我：其實關於這個問題我是比較沒辦法給予一個正確的建議，因為我個人對於薪資的選擇不是那麼很注重，可是依照我在高雄來講，我基本上最低薪資會考慮在 3 萬上下，北部的話…可能要在 4 萬左右，但是基本上如果你不希望自己在談薪資的時候被砍太多，你可以在對方問你期望薪資的時候稍微拉高一點點，讓自己有個彈性空間讓對方可以殺價，例如你希望的期望薪資是 4 萬，所以你就可以考慮談薪資時講 4 萬 2 左右，這樣子你就有兩千塊的空間給對方殺，當然也有公司可能會殺到 3 萬 8，但是基本上許多公司都會三個月試用期過後就會調回四萬，其實對於你的期望薪資上並沒有太大差異，那我個人對於薪資的選擇不是很注重，最主要是因為我會將薪資放在第二順位，第一位則是能不能有機會成長，所以薪資部分我就會壓在生活最低水平，這樣子當你發生資遣 or 你想換工作的時候你才能將現有的開發能量呈現在下一份薪資上。

對於自己目前的工作不滿意

學員：我對於自己目前工作感覺不到挑戰性，而且在成長幅度上非常的平順，感覺自已不會成長了。

我：首先你必須要有一個基本觀念，公司其實是會束縛你技術成長的地方，除非公司今天會常常需要開發新系統又或者它是一間新創公司，需要你開發新系統，否則居多都會是以維護系統為主，那維護系統主要是穩定性，所以也比較不會有事沒事就要你改版升級，如果你想要有挑戰性，那最簡單的方法就是換一間公司，但是通常來講換公司會有一段陣痛期，如果目前自己技術能量不夠的狀況下我也不是很建議你換公司，我反而會建議你下班後多學習一些主流技術，當公司發生狀況時，至少你還保有技術能量，當然維護既有專案也是有好處，你的學習曲線通常會比較不會那麼高，因為絕大部分問題都可以在網路上找到，如果你選擇的是一個潮到爆的技術，反而你的學習曲線會比較高，因為你會比較難找到資料。

結尾

被車輪戰到最後聲音差點啞掉，雖然我不能給予實質上幫助，但我盡可能會給予一定建議，也因為有部分學員跟我講說實在太多人包圍我，導致沒機會問我一些問題，所以如果你還有什麼想與我聊聊的也歡迎你私下與我聊聊，當然可能會回比較慢就是了(汗)

那麼我是 Ray，我們下一次六角同學會見囉 :D
