基本上一般開發上除非專案太大需要判斷的非常多，否則在效能比較上是不太明顯的，如果 if...else if... 太多的話是可以考慮改用 switch，其主要原因在於 if...else if... 寫太多會有很嚴重的巢狀問題，而搞不清楚花括號 { 在哪裡，此時 switch 就會顯得非常好用。
