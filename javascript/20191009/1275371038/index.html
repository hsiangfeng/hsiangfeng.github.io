undefined

身為 JavaScript 開發者 undefined 以及 not defined 是非常常見的，當然如果能夠搞懂這兩者差異，對於自己在實際開發上是有一定幫助，首先我們可以在 MDN 文件中可以看到 MDN 自己都形容 JavaScript 裡面有一個有點奇怪的關鍵字，那個關鍵字是 undefined，那 undefined 到底是什麼?讓我們先從程式碼來了解吧。
