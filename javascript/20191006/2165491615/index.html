取得月份

取得月份的方式是 getMonth() ↓

但是這邊要注意一件事情，getMonth() 從 0 開始計算，所以實際應用時請記得 +1。

  • 一月 = 0,
  • 二月 = 1,
  • 三月 = 2,
  • 四月 = 3,
  • 五月 = 4,
  • 六月 = 5,
  • 七月 = 6
  • 八月 = 7
  • 九月 = 8
  • 十月 = 9
  • 十一月 = 10
  • 十二月 = 11

取得日期

取得日期的方式是 getDay() ↓

日期也一樣要注意getDay() 是從 0 開始計算，所以實際應用時請記得 +1。

  • 星期日 = 0
  • 星期一 = 1
  • 星期二 = 2
  • 星期三 = 3
  • 星期四 = 4
  • 星期五 = 5
  • 星期六 = 6
