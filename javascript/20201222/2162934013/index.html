Image

圖片在網頁開發上是非常常見的元素，舉凡產品列表這區塊就非常的常見，畢竟你一定會上傳圖片，否則你要如何讓使用者知道產品樣子呢？因此我們就會使用到 new Image()

而依照 MDN 說明 new Image() 等同於 document.createElement('img')，因此在 MDN 上就有一段範例是關於 new Image() 寫法
