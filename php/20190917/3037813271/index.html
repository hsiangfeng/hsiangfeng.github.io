結尾

我們可以發現若不透過 Model 來操作資料是一件非常麻煩的事情，而且這些動作也不符合 MVC 觀念，畢竟沒有將 Model 的邏輯給切出來，所以當系統越來越龐大時，就會導致系統維護上的困擾。
