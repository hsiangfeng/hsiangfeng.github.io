構造器

我不清楚為什麼官方會翻譯成構造器，但是簡單來講 ThinkPHP 提供一些包裝好的 SQL 指令方法，最主要可以方便我們做資料操作，而這資料操作是基於 PDO 參數綁定，PDO 可以保護系統被 SQL 注入，所以我們可以不用去撰寫特殊字符轉譯。
