Basic Git Commands

#Git

Intro

This post has been sitting in my EverNote forever, but as part of cleaning EverNote up I’m pulling it out and rewriting it here as a reference — both for me later and for anyone else who needs a quick cheat sheet.

Some commands come with examples; the simpler ones I’ll just show as screenshots.

Basic command-line commands

Before getting into Git commands, you need a basic understanding of command-line commands, otherwise the rest of this will look like alien gibberish.

Change directory: cd [path]

  • cd C:\Users\Ray\Desktop\project

Go up one level: cd .. and go up two levels: cd ../..

List contents:

  • ls

Make a directory: mkdir [folder name]

  • mkdir my-new-folder

Git commands

Basic setup

Check Git version:

  • git version

List Git config:

  • git config --list

Set user name: git config --global user.name "Name"

  • git config --global user.name "Ray"

Set user email: git config --global user.email "your email"

  • git config --global user.email "abc@gmail.com"

Check Git status:

  • git status

Initializing Git + remote Git

Init a local project:

  • git init

Clone a remote project: git clone [Url]

  • git clone https://github.com/hsiangfeng/Pug-Sweetaste.git

Adding files to version control

Stage a file: git add [filename]

  • git add index

Stage all files:

  • git add .

Commit staged changes: git commit -m 'commit message'

  • git commit -m 'first'

Checking status & history

Check status:

  • git status

Show history:

  • git log

Branches

Show all branches:

  • git branch

Create a new branch: git branch [branch name]

  • git branch develop

Switch branches: git checkout [branch name]

  • git checkout develop

Delete a branch: git branch -D [branch name]

  • git branch -D develop

Merge a branch into the current one: git merge [branch name]

  • If you’re on master: git merge develop

Remote Git operations

Clone a remote project: git clone [Url]

  • git clone https://github.com/Ray/Pug-Sweetaste.git

Clone into a specific folder: git clone [Url] [path]

  • git clone https://github.com/Ray/Pug-Sweetaste.git theme/pug

List remotes: git remote

Push local branch to remote: git push [remote-name] [remote-branch-name]

  • git push origin master

Pull and merge remote branch:

  • git pull

Tags

List tags: git tag

List tag details: git tag -n

Add a tag: git tag [tag name]

  • git tag v1.0.0

Add a tag with a message: git tag -am [message] [tag name]

  • git tag -am 'first' v1.0.0

Delete a tag: git tag -d [tag name]

  • git tag -d v1.0.0

Stash

Stash the current working directory: git stash

List stashes: git stash list

Restore the latest stash: git stash pop

Drop the latest stash: git stash drop

Clear all stashes: git stash clear

Undo commands

Undo commands get a bit messier. For a reference, see Backlog’s Git tutorial — log/reset/revert.
