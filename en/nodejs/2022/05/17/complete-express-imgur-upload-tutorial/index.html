The Complete Express + Imgur Upload Tutorial

#Express #Imgur #Image Upload

Intro

Imgur is a really handy online photo album service with free image hosting. It’s great for upload practice during dev, or even as your blog’s image library. So here’s a tutorial on Imgur image upload with Express.

Note
This series is about Imgur, but after May 16, 2025, Imgur has blocked Taiwanese IPs from uploading images. If you’re in Taiwan, consider using another image upload service.

Imgur prerequisites

Create an Imgur Album

First, learn how to create an Album — later, all uploaded images will be sorted into a specific Album.

Imgur used to let you create Albums directly, but they later hid the button and made it pretty hard to find. 80%+ of people can’t find it, so follow these steps exactly.

Log in to Imgur. After login, click “images” in the user menu.

Find the “All images” dropdown — there’s a “New Album”. Click it.

A modal pops up. Fill in the Album settings and click Save.

After creating, you’ll see an empty page because we haven’t added any images.

Album created.

Great Imgur UX.

Get the Album ID

We need the Album ID later to upload images. Click “gallery profile” in the user menu.

Find “Albums” on the right.

Click “Posts” → “All”. The first one is the Album you just created.

Open the Album and check the URL — only the part in the red box is the Album ID. Remember it, we’ll use it in Express.

Register an Imgur application

Now register an Imgur application — we need client_id and client_secret for upload. Click the link below:

Register an Imgur application

You’ll see this:

Required fields:

  • Application name
    • Any name
  • Authorization type
    • Choose “OAuth 2 authorization without a callback URL”
  • Email
    • Contact info

PS: If you pick “Anonymous usage without user authorization”, the “Authorization callback URL” needs to be https://imgur.com.

After applying, you’ll see this. Save your client_id and client_secret — especially client_secret, don’t lose it (if you do, you have to regenerate it).

Application done. If you lose client_secret, you can go to Applications and click “generate new secret”.

Get an Imgur Refresh Token

We have client_id and client_secret. The last key piece is the Refresh Token. Imgur’s Token has a time limit — to auto-refresh, you need a Refresh Token.

Getting it is unusual — replace ${Client ID} in this URL with the one you just got:

Paste it into your browser address bar and hit Enter.

You’ll hit a warning screen. As long as you confirm this is the app you want, click Allow.

After allowing, you’ll be redirected to Imgur. Copy the current URL — it’ll look like:

We only need the Refresh Token. Save it.

Imgur final check

If you’ve followed along, you should have:

  • An empty Album
  • Album ID
  • Client ID
  • Client Secret
  • Refresh Token

All five are required. If anything’s missing, go back through the steps.
