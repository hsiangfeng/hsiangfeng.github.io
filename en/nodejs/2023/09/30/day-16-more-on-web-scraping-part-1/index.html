More scraping

Intro

This post is also about web scraping, but we'll go after some more complex data.

More scraping

In the previous post we did a demo with my blog. Here's the previous example code:

js
// index.js
const fs = require('fs');
const cheerio = require('cheerio');

const getData = async (url) => {
  try {
    const response = await fetch(url);
    const data = await response.text();
    return data;
  } catch (error) {
    console.log(error);
  }
}

const crawler = async () => {
  const html = await getData('https://israynotarray.com/');

  const $ = cheerio.load(html);

  const postTitleLink = $('.post-title-link');

  const data = [];

  postTitleLink.each((index, element) => {
    const title = $(element).text();
    const url = $(element).attr('href');
    data.push({
      title,
      url,
    });
  });

  fs.writeFileSync('./data.json', JSON.stringify(data));
}

crawler();

That's a basic, simple scraper. But what if today we want to scrape something different?

Different data

I mentioned earlier that scraping is about getting HTML data, parsing it to extract what we want, and storing it in our preferred format. This data can have huge uses — data analysis, visualization, etc.

The thing I think is well-suited as a demo is...

"iThelp Iron Man Contest articles"

Why iThelp Iron Man Contest articles? Because the Iron Man Contest never provides a reading rankings feature — so we can try implementing it.

Pre-scraping prep

Before starting, plan our requirements:

  • Contestant name
  • Contest category
  • Article title
  • Article URL
  • View count
  • Like count
  • Comment count

Roughly these five pieces of data. The data source? Of course it's the 2022 Iron Man Contestant List.

Note Why 2022 Iron Man? Because the 2023 Iron Man hasn't ended yet, so we use 2022. The concept is the same.

If no problem, set up the project:

bash
mkdir example-ithelp-crawler
cd example-ithelp-crawler
npm init -y
npm i cheerio
touch index.js

(Don't forget to add the start command to package.json.)

Now of course we have to start...

Interrupting

Hold on — remember the four steps from the previous post?

  • Analyze the source
  • Get the data
  • Parse the data
  • Store the data

Step one is analyze the source — otherwise how would you know what to scrape? Open 2022 Iron Man Contestant List in your browser. We can see lots of contestant info on this page, where we can extract:

  • Contestant name
  • Contest category
  • Article title
  • Article URL

You'll notice the contestant data structure is all under .list-card. So we can use .list-card to get contestant data — pretty clear target. Let me paste the code so you can see the final result. We won't use fs to save files yet:

js
// index.js
const cheerio = require('cheerio');

const getData = async (url) => {
  try {
    const response = await fetch(url);

  }

const crawler = async () => {

We can see the last page is 56. We can use this number to decide how many pages to scrape, then use a for loop to scrape each page. Here's the code.

It might be a bit complex, but I'll add comments to every line for clarity:

js
const fs = require('fs');

const cheerio = require('cheerio');

const getData = async (url) => {
  try {
    const response = await fetch(url);
    const data = await response.text();
    return data;
  } catch (error) {
    console.log(error);
  }
}

const crawler = async () => {
  // First, get the first page's data
  const html = await getData('https://ithelp.ithome.com.tw/2022ironman/signup/list');
  // Parse the html with cheerio
  const $ = cheerio.load(html); 
  // Get the last page number element
  const paginationInner = $('span.pagination-inner > a').last();
  // Extract the text and convert page number text to number
  const lastPage = Number(paginationInner.text());
  // Create an empty array to hold the data
  const data = [];

  // Use a for loop to scrape each page
  for(let i = 1; i <= lastPage; i++) {
    console.log(`Scraping page ${i}`);

    // Scrape each page's data
    const html = await getData(`https://ithelp.ithome.com.tw/2022ironman/signup/list?page=${i}`);

    // Parse with cheerio
    const $ = cheerio.load(html);

    // Contestant cards
    const listCard = $('.list-card');

    // Use each to scrape each contestant's data
    listCard.each((index, element) => {
      // Get name, category, title, URL
      const name = $(element).find('.contestants-list__name').text();
      const category = $(element).find('.tag span').text();
      const title = $(element).find('.contestants-list__title').text();
      const url = $(element).find('.contestants-list__title').attr('href');
      data.push({
        name,
        category,
        title,
        url,
      });
    });

    // Avoid excessive requests overloading the server
    await new Promise((resolve) => {
      setTimeout(() => {
        resolve();
      }, 5000); // Run once every 5 seconds
    })
  }

  // Write data to data.json
  fs.writeFileSync('./data.json', JSON.stringify(data));
}

crawler();

After running this, you'll have data for all contestants.

Note You'll see I wrote "Avoid excessive requests overloading the server" code — mainly to prevent overload. We use setTimeout to throttle to once every 5 seconds.
