2019-05-15 Hexo

Useful NexT Settings and Features

#Hexo #NexT

Intro

A list of the tweaks I’ve made to NexT and a few good plugins I’ve installed.

This post covers a lot of polish and feature additions for NexT (including small details).

Most config edits below are in theme/NexT/_config.yml. If we jump to the main system config, I’ll call it out.

Environment for this post

Hexo version

  • hexo: 3.8.0
  • hexo-cli: 2.0.0

NexT version

  • NexT: 7.1.0

Copyright notice

The footer block at the bottom of the page.

Around line 65, search for copyright. If you leave it empty, NexT defaults to the site author name.

One side note: if you don’t want the “Powered by Hexo v3.8.0 | Theme – NexT.Pisces v7.1.0” line, change enable to false on lines 67 and 73.

Bonus: you can add a small animation to the icon next to the copyright.

Set animated to true.

Polishing code blocks

Default code blocks have square corners.

Via _config.yml, you can round the corners and add a copy button.

Search for codeblock. Settings here mainly tweak the copy button:

  • border_radius — corner radius (default 1)
  • copy_button
    • enable: true — enables the copy button
    • show_result — shows copy result
    • style — only blank or flat (default looks rough)

To change the code block colour scheme, search for highlight_theme. Options are: normal | night | night eighties | night blue | night bright.

See the Tomorrow theme examples.

Adding a bookmark feature to NexT

A nice feature — next time the user visits, they can jump back to the article they were last reading.

In your terminal, cd into the NexT theme folder:

Run:

Open _config.yml, find bookmark, and set enable to true.

A bookmark icon will appear in the top right corner.

If I close the browser while reading “NexT’s useful features”, next time I click the bookmark and it sends me back to that article.

Enable article sharing

Note: this feature was removed and moved to a plugin in around NexT 7.2.0.

Useful feature, off by default. Once enabled, every article gets share buttons at the bottom, and you can customise which ones appear.

Search _config.yml for Likely Share and set enable to true. You can customise the share button names; to hide a button, just add # in front.

Lazy load images

cd into the NexT theme folder:

Run:

In the main _config.yml (not the theme’s), add this at the bottom:

  • onlypost — false means everything (including avatar) gets lazy-loaded
  • loadingImg — placeholder shown until the actual image loads. Drop the placeholder in theme/source/images

Adding a comments system

Register and log in at Disqus, then click GET STARTED.

Pick I want to install Disqus on my site.

Fill in:

  • Website Name — custom; Hexo will use this later
  • Category
  • Language

I’m creating one for testing here.

Open NexT’s _config.yml, find Disqus, paste the Website Name into shortname:

  • enable — toggle comments
  • shortname — Website Name from above
  • count — show post comment counts

Comments will now show at the bottom of every article.

To view or manage comments: Disqus admin.

Cleanup

To remove a cloned feature, just delete the relevant folder under hexo-theme-next-master\source\lib.

Warning: double-check the folder name before deleting, so you don’t accidentally remove an important plugin.
