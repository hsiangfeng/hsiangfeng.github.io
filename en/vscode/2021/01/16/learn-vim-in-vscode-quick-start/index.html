Intro

This is a Vim beginner walkthrough. Fair warning — you'll definitely hit walls early on. The concept is actually pretty close to learning a new input method.

Installing the extension

The fastest way to get used to Vim is to switch your existing editor into Vim mode, so install this extension first:

VSCode Vim extension

(I'm using VSCode to write this.)

Once installed, if you've never touched Vim before, you'll feel like a complete beginner — so this post is a basic walkthrough.

This walkthrough is based on the VSCode extension, so a few things might differ slightly from real Vim, but in my experience it's pretty close. Maybe because I've used Vim a bit before.

Modes

Vim has four basic modes:

  • Normal mode
    • Also called the default mode.
  • Insert mode
    • Also called the input mode — text-typing mode.
  • Visual mode
    • Used for selecting text or highlighting for emphasis. Personally I mostly use it for selecting.
  • Command mode (start with :)

When you switch modes, the bottom of VSCode shows you which mode you're in.

Mode indicator (happened to be listening to music)

A lot of actions only work in specific modes. The most common newbie problem is "I don't know how to get out of this mode." Easy — press ESC and you're back in Normal mode.

Moving around (Normal mode)

Normally you'd move with the arrow keys. In Vim, you stay in Normal mode and use h, j, k, l to move (arrow keys still work too, no judgement).

(If you're not sure which mode you're in, just spam ESC.)

KeyWhat it does
hMove left one character
jMove down one line
kMove up one line
lMove right one character

Note: case matters.

Insert mode

Once you've moved the cursor where you want it, you'll want to start typing. That means entering Insert mode. There are four ways:

KeyWhat it does
iEnter Insert mode before the cursor
aEnter Insert mode after the cursor
oOpen a new line below and enter Insert mode

Press i

Press a

Press o

Exit Insert mode with ESC (as always — if you want to exit anything, just spam ESC). I'd also recommend building the habit of hitting ESC after every action, otherwise you'll end up in the classic situation:

I've been learning Vim for ages and I still don't know how to exit. (lol)

Selecting, copying, pasting (Visual mode)

Now the basics — select, copy, paste. You could fall back to the standard Ctrl + C / V, but you're using Vim now, you might as well commit.

Enter Visual mode by pressing v. The cursor doesn't look any different visually, but the moment you use the arrow keys you'll see text being selected.

Visual mode

Once in Visual mode:

KeyWhat it does
yCopy (yank) selected text
pPaste the last copied text
dCut (delete) selected text
uUndo the last action (e.g. delete or cut)

Reminder: case still matters.

Movement tricks

Vim has a really fun command-mode feature for jumping to a specific line number, which I love. Say you're 100+ lines into a file and need to jump to line 1. Press Shift + : and the bottom bar shows :|.

Command mode prompt

Type the line number, hit Enter, and you jump there.

Jump to line

You might want a faster way to jump to the top or bottom of the file without typing commands:

KeyWhat it does
ggJump to the first line of the current window (press g twice — like saying "gg" in a game when you've died and need to restart)
Shift + gJump to the last line (with Shift, g becomes uppercase G — I'm using lowercase in the table to show the keystroke)

Note: both of these only work in Normal mode.

You might also want to jump to the start or end of the current line (not the window). Three ways:

KeyWhat it does
0Jump to the start of the line (the digit 0, not the letter O)
Shift + 6 (^)Jump to the start of the line
Shift + 4 ($)Jump to the end of the line

All three require Normal mode.

One last navigation trick — search. Press / to enter search mode, then type what you want to find (English only, no Chinese).

After typing, press Enter to commit the search. If the result isn't quite what you wanted, use these to jump between matches:

KeyWhat it does
/Enter search mode (English only, press Enter to confirm)
Shift + 8 (*)Jump to next match
Shift + 3 (#)Jump to previous match

Found what you wanted? Press i or a to drop into Insert mode.

Wrapping up

Vim genuinely has a learning curve at the start. This post is just a basic intro — there are a ton of more advanced tricks I haven't covered. If you've ever learned an alphabet-key input method like Boshiamy, you'll recognise the feeling: "wow there are a lot of key roots to memorise." Vim is the same. But once you're comfortable, moving around to edit code or write articles becomes really fast.
