RE: Learning JavaScript From Zero — EP.0: The Table of Contents (and Why I Started)

#JavaScript #RE：從零開始的學習 JS 生活 #第 11 屆 iT邦幫忙鐵人賽

Intro

This is the second series I wrote for the iThome Ironman challenge, and it’s mostly about JavaScript — but I’m leaning heavily toward beginners here. I mean, the whole thing is literally called “from zero,” so that’s kind of the point.

The “isekai” reboot

This was actually my first time entering the Ironman challenge, but it was my second Ironman series topic. My first one was Learning ThinkPHP From the Basics, which was mostly backend stuff.

While I was writing the ThinkPHP series, I started wondering whether I should torture challenge myself a bit more. But I was worried that writing two series at the same time would tank the quality of both. After a few days of thinking it over, I decided to just write down my own beginner notes from when I was learning JavaScript. So the content here is pretty beginner-friendly — which is also why I named the series RE: Learning JavaScript From Zero. And if I get anything wrong, please go easy on me.

The main reason I wanted to do it this way is that I wanted an excuse to go back and revisit the JavaScript fundamentals. The more stuff I pick up, the more my brain turns into a goldfish brain, so I figured I’d take this chance to write down everything from when I first got into JavaScript. That way, when my memory inevitably fails me, I can at least flip back through these articles.

Starting from the next post, I won’t put the specific topic in each title — I’ll just use the day as a label instead. But I’ll do my best to mention what the next post covers at the end of each article.

And of course, this whole series will also be posted on my blog → It’s Ray, not Array.
