Gemini Gems Tutorial: Easily Build Your Own AI Assistant

#AI #Gemini #Gemini Gems #AI Assistant

Intro

Have you ever thought about how convenient it would be to build your own AI assistant — one that responds to your specific needs? This post walks you through Google’s Gemini Gems service, step-by-step, so you can build your own AI assistant.

What is Gemini Gems?

Gemini Gems is an extra service Google has added on top of Gemini. You can think of it as an AI platform where you build your own AI experts/assistants without having to learn anything about AI models.

If you’ve used OpenAI’s GPTs, this is going to feel really familiar — the concept and the way it works are pretty much the same. The biggest difference is the ecosystem. You can easily pull various Google services into your Gem as reference, which boosts what it can do significantly. You can even let your Gem use Google Search to find the answers you need.

How to use Gemini Gems

It’s really easy to use. Head over to Gemini and click “Gem”.

Scroll down and you’ll find a section called “My Gems”, with a “New Gem” button next to it. Click that.

You’ll land on the Gem creation screen. There are a few fields:

  • Name: the name of this Gem — this is what shows up on screen.
  • Description: just a quick intro to what this Gem is for, so users understand its purpose.
  • Instructions: this is the most important part — it’s the Prompt. It’s how you interact with the Gem. You tell it what you want it to do, or how you want it to help you, and the Gem responds based on your instructions.
  • Default tools: which tools are enabled by default when someone uses your Gem — Deep Research, image generation, guided learning, etc. If you’re building an image-generation helper, you’d default it to “image generation”.
  • Knowledge: reference material. You can upload files, photos, files from Google Drive, or even upload GitHub code.

  • Name: Chinese-English Translation Helper
  • Description: A Gem that helps you translate between Chinese and English. Type Chinese and it translates to English, type English and it translates to Chinese.

  • Default tools: we won’t enable any tools here, since the translation is handled by the Gem itself.
  • Knowledge: we won’t provide any reference material here either, since translation is a basic capability that doesn’t need additional info.

After you’ve filled all this in, you can see a live preview on the left, so you can get a feel for what your Gem will look like and test it on the spot.

If everything looks good, hit “Save” and your Gem is done.

Pretty easy, right? Sharing it with friends is easy too — just click “Share” and change the general access to “Anyone with the link”, and you can send the link to your friends.

That’s the whole flow for building your own AI assistant — quick and simple. One last reminder though: to avoid AI hallucinations, when you send a message, switch the mode from “Fast” to “Thinking” or “Pro”. This significantly reduces hallucination issues.

Common gotchas with Gems

Building Gems is easy, but there are a few limits worth knowing about. Here are some common gotchas to avoid:

Sandboxed environment, no outbound HTTP requests

Gems run in a sandbox, which means they can’t make outbound HTTP requests. If you were hoping to have your Gem call external APIs (like fetching real-time stock prices or weather), that’s not going to work. The only way it can pull info from the web is through the Google Search tool — it can’t hit APIs directly.

Can’t execute code to render charts

Gemini itself has code execution, but inside Gems you can’t run drawing-related code (like using Python’s matplotlib to plot a chart). If your Gem needs to generate charts or visualizations, it can only describe them in text — it can’t generate the actual image.

Prompt instructions aren’t strictly followed

Like with GPTs, the instructions (Prompt) you give your Gem aren’t absolute rules — the model sometimes ignores or drifts away from your instructions, especially when the instructions are complex or vague. The fix is to write your Prompt as specifically and clearly as possible, and add a few examples to guide the model’s behavior.

No automation hooks for external services

Gems currently don’t expose Webhooks or APIs, so you can’t connect them to external services the way GPTs Actions can (auto-send emails, write to a database, etc.). They’re better suited as conversational assistants than as automation tools.

Wrap up

To wrap up, I’ll share a Gem I built — RayStock AI Stock Analysis. Just type in the stock ticker you want to analyze and it generates an analysis of the fundamentals, technicals, and capital flow, with a quick summary report so you get a feel for the stock’s situation faster.

Same reminder — when using it, don’t forget to switch to “Thinking” or “Pro” mode!
