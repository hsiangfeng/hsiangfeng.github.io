Learning Python From JavaScript (30): Building a Web Scraper

#JavaScript #Python #Learning Python From JavaScript #iThome Ironman Contest (13th)

Intro

We’ve picked up a fair amount of Python by now, and this also happens to be the last day of the Ironman contest. So let’s put it all together and write a simple scraper for the iThome Ironman site.

What we’re building

As usual, let me quickly lay out what we’re actually doing here.

We’ll mainly use the Requests and BeautifulSoup packages to build a few things. As for what those things are, let’s just look at the requirements:

  • Scrape the Ironman homepage to grab topic info, with these fields:
    • Topic URL
    • Topic title
    • Number of contestants
  • Use that topic info to pull contestant data for each topic, with these fields:
    • id
    • Contestant name
    • Ironman series title
    • Contestant’s Ironman link
    • “Knock count” (likes / waiting-for-more count)

Target URL: 2021 iThome Ironman homepage
