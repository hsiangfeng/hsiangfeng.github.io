#JavaScript #Python #Learning Python From a JavaScript Angle #13th iThome Ironman Contest

Intro

We’re heading into the home stretch of the Ironman contest, so this post covers a really common package: BeautifulSoup. It’s one of the most-used tools for web scraping in Python, so let’s go learn how to use BeautifulSoup.

BeautifulSoup

Python gets used a lot for web scraping, and the package most people reach for is BeautifulSoup. So… what exactly is BeautifulSoup?

Let me just borrow Wikipedia’s description here:

Beautiful Soup is a Python package for parsing HTML and XML documents, including ones with malformed markup like unclosed tags (these are often called “tag soup”). It creates a parse tree for the page that can be used to extract data, which is super handy for web scraping.

From that we can roughly gather that BeautifulSoup mainly parses the HTML files we download, and using it makes grabbing specific chunks of data way easier.

Oh, by the way — if you’ve ever looked up BeautifulSoup, you’ll have noticed its cover image is a medieval-style picture, and a pretty distinctive one at that:

The name BeautifulSoup comes from a poem in Alice’s Adventures in Wonderland, which is why some translated docs even call it the “Alice document.” That’s also why the package’s cover image is always medieval-style — after all, Alice in Wonderland is set in medieval Europe (if I remember right).

That was just a little fun aside about BeautifulSoup. Now let’s get on with installing it!
