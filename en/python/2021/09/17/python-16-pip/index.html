Learning Python From a JavaScript Angle (16) - pip

#JavaScript #Python #iThome Ironman Contest #Learning Python From a JavaScript Angle

Intro

We’ve covered a lot of Python’s basic syntax in the earlier chapters, so now I want to take a detour and talk about pip. It’s something you’ll reach for constantly in real-world development.

pip3

Some people type pip, others type pip3. From here on I’ll just call it pip3 to mean Python 3, so we don’t get confused.

So what is pip3? In short, it’s a package manager. If we swing the camera back over to JavaScript — if you’ve ever used Vue CLI, Webpack, Gulp, and so on, then you’ve used npm to install packages. So you can basically think of pip3 as npm (Ruby’s is gem, if I remember right — correct me if I’m wrong).

(If you’re not comfortable with npm commands, you can read this post (article in Mandarin) for a quick rundown, so the rest of this doesn’t trip you up.)

pip3 is honestly really, really important, because during development we constantly need to install third-party packages (the non-official, non-built-in stuff). With npm, it gets installed alongside Node.js. What about Python? Pretty much the same deal — Python 3, and Python 2.7.9 through 2.8.0 (I think), ship with pip3 built in.

So how do you check whether your machine has pip3? Just run the command below (since this post is about Python 3, you’d type pip3; if you’ve tweaked things, it might just be pip):

Older versions (like Python 2.x) use pip --version.

And that’s the simplest way to tell whether your machine has the pip3 package manager. Easy enough, right? Congrats, you just learned one more thing:

Common pip3 commands

Sure, we can run pip3 --help in the terminal to see every single pip3 command, but honestly — are we ever going to use all of them? Nope. Even with npm, the commands I actually use boil down to a handful. So here’s a little table of the ones you’ll reach for most:

Description Command
Install a package pip3 install [ package-name ]
Remove a package pip3 uninstall [ package-name ]
List installed packages pip3 list
Output installed package info pip3 freeze
pip3 command help list pip3 help | pip3 -h | pip3 --help
Search for a package pip3 search [ package-name ]
pip3 version info pip3 --version | pip -V

Let’s actually play around with these.

(Friendly heads-up: if pip3 doesn’t work for you below, switch to pip.)
