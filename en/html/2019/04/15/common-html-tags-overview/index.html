Intro

There are a ton of HTML tags. Here's a cheat-sheet covering the common ones, with quick explanations.

HTML tags

Comments

Comments in HTML are visible from DevTools. Hit F12 and you'll find what I hid in this HTML.

HTML
<!-- -->

Line break

Just a line break.

HTML
<br>

Headings

Headings use h tags. There are six levels:

HTML
<h1>This is H1</h1>
HTML
<h2>This is H2</h2>
HTML
<h3>This is H3</h3>
HTML
<h4>This is H4</h4>
HTML
<h5>This is H5</h5>
HTML
<h6>This is H6</h6>

Note: there should only be one h1 on a page, otherwise search-engine crawlers won't know which is the most important heading.

Hyperlinks

Hyperlinks default to blue underlined text, though my blog uses a different colour.

Example: A link

HTML
<a href="#" target="">A link</a>

target has four values but I'll just cover the two common ones.

Open in a new tab:

Example: Open in new tab

HTML
<a href="#" target="_blank">Open in new tab</a>

Open in same tab (default):

Example: Open in same tab

HTML
<a href="#" target="_self">Open in same tab</a>

Abbreviation

Most commonly used for English abbreviations.

Example: Hi MyName, that's my name.

HTML
Hi <abbr title="Xiaoming Wang">MyName</abbr>, that's my name.

Address

Marks the content as an address — renders in italic.

Example: I live in

Taiwan
HTML
I live in <address>Taiwan</address>

Media

The media tags were added in HTML5. Use them for audio/video playback.

Audio

Example:

HTML
<audio src="http://www.w3school.com.cn/i/horse.ogg" controls>Your browser doesn't support HTML5</audio>

audio supports four key attributes.

Show playback controls:

HTML
<audio src="http://www.w3school.com.cn/i/horse.ogg" controls>Your browser doesn't support HTML5</audio>

Auto-loop after finishing:

Example:

HTML
<audio src="http://www.w3school.com.cn/i/horse.ogg" loop controls>Your browser doesn't support HTML5</audio>

Autoplay:

Embedding this example would be annoyingly loud, so just the code:

HTML
<audio src="http://www.w3school.com.cn/i/horse.ogg" autoplay controls>Your browser doesn't support HTML5</audio>

Preload (start loading on page load):

HTML
<audio src="http://www.w3school.com.cn/i/horse.ogg" preload controls>Your browser doesn't support HTML5</audio>

Autoplay + loop, no controls visible:

HTML
<audio src="http://www.w3school.com.cn/i/horse.ogg" controls loop autoplay>Your browser doesn't support HTML5</audio>

Video

Same concept as YouTube playback...

Example:

HTML
<video src="http://www.w3school.com.cn/i/movie.ogg" controls>Your browser doesn't support HTML5</video>

Attributes are mostly the same as audio, but video supports width/height.

Example:

HTML
<video src="http://www.w3school.com.cn/i/movie.ogg" controls width="100px" height="100px">Your browser doesn't support HTML5</video>

Autoplay + loop:

HTML
<video src="http://www.w3school.com.cn/i/movie.ogg" controls loop autoplay>Your browser doesn't support HTML5</video>

Button

You'll see this on signup forms, login forms, basically everywhere.

Example:

HTML
<button type="button">A button</button>

button has three types.

Default button (no function):

HTML
<button type="button">A button</button>

Submit button — pairs with forms:

HTML
<button type="submit"></button>

Reset button:

HTML
<button type="reset"></button>

Layout

The main layout tag is <div>:

HTML
<div>...</div>

Horizontal rule

Example:

HTML
<hr>

Paragraph

Wrap any block of text in p.

Example:

This is an article. The text goes on. And on. And on. And on and on.

HTML
<p>
This is an article.
The text goes on.
And on.
And on.
And on and on.
</p>

Superscript and subscript

Common in chemistry and math notation.

Superscript: Co2

Subscript: 24

HTML
2<sup>2</sup>
2<sub>2</sub>

Lists

Lists are common for menus, features, product listings. Two main types: ul (unordered) and ol (ordered).

ul example:

  • Item 1
  • Item 2
  • Item 3
  • Item 4
HTML
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
  <li>Item 4</li>
</ul>

ol example:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
HTML
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
  <li>Item 4</li>
</ol>

Tables

Tables are used to display tabular data. The structure is table wrapping thead, tbody, and tfoot. tr defines a row. td defines a cell — content goes here. th is a header cell, centred by default — use for header text.

I can't show this inline cleanly so just the code:

HTML
<table>
  <thead>
    <tr>
      <th>Heading 1</th>
      <th>Heading 2</th>
      <th>Heading 3</th>
      <th>Heading 4</th>
      <th>Heading 5</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Cell 1</td>
      <td>Cell 2</td>
      <td>Cell 3</td>
      <td>Cell 4</td>
      <td>Cell 5</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Table footer</td>
      <td>Table footer</td>
      <td>Table footer</td>
      <td>Table footer</td>
      <td>Table footer</td>
    </tr>
  </tfoot>
</table>

The web "three musketeers" (forms)

Form elements are sometimes called the "three musketeers" of web pages. There's a lot to cover.

form

HTML
<form action="" method="post">
            
</form>

action is where the form sends data to; method is how — post or get.

label

Used to define labels for input fields. Main UX benefit: clicking the label focuses the corresponding input.

HTML
<label for="account">account</label>

Main attribute is for, which matches an input's id.

input

input has lots of attributes — I'll just cover common type values.

Button input:

HTML
<input type="button" value="This is an input button">

Checkbox:

HTML
<input type="checkbox">

Colour picker:

HTML
<input type="color">

Date picker:

HTML
<input type="date">

Email:

HTML
<input type="email">

File upload:

HTML
<input type="file">

Hidden input (you can't see it, that's the point):

HTML
<input type="hidden">

Number input:

HTML
<input type="number">

Password:

HTML
<input type="password">

Radio:

HTML
<input type="radio">

Range slider:

HTML
<input type="range">

Reset:

HTML
<input type="reset" value="Reset">

Search:

HTML
<input type="search">

Submit:

HTML
<input type="submit">

Tel:

HTML
<input type="tel">

Plain text:

HTML
<input type="text">

Time:

HTML
<input type="time">

Combining the three musketeers (form + label + input)

Heads up — for radio buttons to be exclusive, give them a shared name. The form below is just for show, it doesn't send data anywhere.

Yes No
HTML
<form action="#" method="post">
  <label for="account">account</label>
  <input type="text" id="account">
  <label for="password">password</label>
  <input type="text" id="password">
  <label for="password">Do you eat meat?</label>
  <input type="radio" name="gender"> Yes
  <input type="radio" name="gender"> No
  <label for="tel">Phone</label>
  <input type="tel" id="tel">
  <input type="submit">
  <input type="reset" value="Reset">
</form>

Multi-line input

A normal text input is single-line. For multi-line input, use a different tag.

Example:

HTML
<textarea rows=5 cols=10>
Multi-line input — you can break lines here.
</textarea>

Wrapping up

There are a lot of HTML tags out there and you won't use them all. The ones above cover 70–80% of what you'll actually need. Look up the rest as you go.
