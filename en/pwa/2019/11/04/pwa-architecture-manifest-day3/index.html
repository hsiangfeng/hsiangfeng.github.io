Intro

Time to dig into PWA's architecture. I'm splitting this into two posts.

Web App Manifest

PWA mostly relies on two files: manifest.json and the Service Worker. This post focuses on the Web App Manifest.

manifest.json is a JSON file. It contains the app name, icons, app URL, etc. — basically the metadata PWA needs.

Here's the manifest template from Google's docs:

json
{
  "short_name": "Maps",
  "name": "Google Maps",
  "icons": [
    {
      "src": "/images/icons-192.png",
      "type": "image/png",
      "sizes": "192x192"
    },
    {
      "src": "/images/icons-512.png",
      "type": "image/png",
      "sizes": "512x512"
    }
  ],
  "start_url": "/maps/?source=pwa",
  "background_color": "#3367D6",
  "display": "standalone",
  "scope": "/maps/",
  "theme_color": "#3367D6"
}

JSON field reference

FieldWhat it doesRequiredNotes
short_nameShort name of the appYes
nameFull name of the appYesIf short_name is set, it takes priority over this
iconsApp iconsYesArray. Recommended to provide multiple sizes; the example here uses Google's official sample. PNG is recommended
start_urlApp entry URLYesUsually ./index.html
background_colorApp background colourYesThe background colour is the background colour (got hit)
displayDisplay mode — fullscreen, standalone, minimal-ui, browserNoDefault is standalone. Most people don't change this unless there's a specific need
scopeApp scopeNoRarely tweaked
theme_colorApp theme colourYesMostly applied to the mobile status bar and window chrome
descriptionApp descriptionNoJust a short description of the PWA
dirReading direction — ltr (left-to-right), rtl (right-to-left), autoNoRarely set, but noting it for completeness
langApp languageNoIf unset, PWA falls back to the system default
orientationScreen orientation — any, natural, landscape, portrait, portrait-primary, portrait-secondary, landscape-primary, landscape-secondaryNoAlso rarely set

The trickiest field is probably theme_color. Google has an example image — the theme colour is the blue area in the screenshot.

App theme colour

One more thing worth mentioning — the PWA splash screen. It's composed of three manifest fields:

  • name
  • background_color
  • icons

So design a nice icon and the splash screen auto-generates — no extra work.
