The Complete Mac Terminal Setup: iTerm2 + Zsh + Oh My Zsh + Powerlevel10k

#Terminal #macOS #Software

Intro

This post walks through how I customize my Mac terminal — I’ll go step by step since this stretch has a few moving parts.

Why customize the terminal at all?

You can develop without a customized terminal — it’s not critical. But a few reasons I do it:

  • Surface useful info at a glance — current branch, path, status
  • The default macOS Terminal is just kinda ugly to stare at all day
  • Honestly it’s also just satisfying

Sounds soft, but the before-and-after is real. Setup process has a few steps though.

Install iTerm2

Next — iTerm2. iTerm2 is also a terminal — why install a separate one when macOS has Terminal built in? Because the built-in Terminal has hard limits on customization. iTerm2 is much more flexible.

iTerm2 color theme

iTerm2 has built-in color presets. iTerm2 → Settings → Profiles → Colors → Color Presets:

If none of those work for you, iTerm2-Color-Schemes has a huge library.

Pick a scheme, download as a zip, or grab the individual file directly from:

https://github.com/mbadolato/iTerm2-Color-Schemes/tree/master/schemes

Search for the scheme name. I like Atom — click into Atom.itermcolors, click Raw, save the file.

Back in iTerm2: Settings → Profiles → Colors → Color Presets → import → pick the file. Done.
