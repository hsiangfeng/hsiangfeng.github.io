Common Discord Problems and How to Fix Them

#Discord #Troubleshooting

Intro

This post collects common Discord problems and their fixes, in the hope that future-you (or someone Googling around) can find an answer.

What is Discord, anyway?

Discord started life as voice software for gamers, but over time it’s added a ton of features and gradually turned into a general-purpose community platform. The flip side is that all those features come with their own quirks, so let me run through the ones I see come up the most.

If you’re new to Discord, these earlier posts might help:

  • Discord beginners’ complete guide
  • Discord quick start: a hands-on walkthrough

Can’t log into Discord

There are basically two reasons your Discord account can’t log in:

  • Account is locked
  • You forgot the password

If it’s just a forgotten password, reset it and you’re good. But a locked account is a different story — accounts usually get locked for violating Discord’s rules, most commonly using bots to mass-DM people or to mass-join servers. Both are against Discord’s terms. If your account is locked, your only real option is to write to Discord support and hope they unlock it.

Can’t accept an invite

This usually shows up when you click an invite link but can’t join. A few possible reasons:

  • You’re banned from that server
  • The server has join restrictions

If you’re banned, you basically have to DM or somehow contact the server’s admin. People generally get banned for breaking server rules — most commonly because their account got hijacked. If you didn’t break the rules and you didn’t get hijacked… congrats, you’re just disliked.

If it’s a server restriction, you need to check what verification level the server has. Discord offers these verification levels:

They are:

  • Low → must have a verified email
  • Medium → verified email + account older than 5 minutes
  • High → verified email + account older than 5 minutes + member of the server for 10+ minutes
  • Highest → verified email + account older than 5 minutes + member of the server for 10+ minutes + verified phone number

Most servers will have at least “Low” enabled. If you’re a brand-new account, Discord is probably already nagging you to verify your email. If you’re an older account, double-check that your email actually got verified at some point — if not, verify it first, then try the invite again.

This verification level also affects whether you can send messages in the server.

Note
The reason this feature exists is to filter out ad bots. Servers that disable it for user convenience tend to see their community health drop fast, so most Discord communities keep it on.

How do I leave a voice channel?

Sometimes you’ll accidentally drop into a voice channel while clicking around a server. You’ll see this icon in the bottom left:

To leave, just click the hang-up icon and you’ll disconnect.

Side note: if you want to lurk without talking, click the mic icon at the bottom to mute yourself.

(There’s also a deafen option, but at that point you might as well just leave the channel.)

No audio / can’t hear anyone / static

If you use Discord voice a lot, you’ll run into this eventually. It usually happens when you join a voice channel and then plug in a headset/mic, and Discord doesn’t pick up the new input device — suddenly nobody can hear you or you can’t hear them.

The fix is simple. Open Discord settings.

Go to “Voice & Video”.

Switch your input/output device to something else and back.

That forces Discord to re-detect your mic/headset. If that doesn’t work, the issue is probably with your hardware.

Can’t DM a specific person

This is usually because the user has tweaked their Privacy & Safety settings. In User Settings → Privacy & Safety, there’s a toggle called “Allow direct messages from server members”.

If they have it off, you need to be friends first before you can DM them. If it’s on, you can DM directly.

Stranger DMs disappeared

Discord later added a feature called “Message Requests” — DMs from non-friends get funneled into this section instead of showing up in your main DM list.

It’s similar to Facebook’s “Message Requests” — reply or hit “Accept DM” and the message moves into your main DM list. The goal is to cut down on ad-bot spam.

If you don’t want this feature, go to User Settings → Privacy & Safety → “Enable message requests from server members you may not know” and turn it off. Then stranger DMs go straight to your main list.

Text or emoji not showing up

If text, emoji, or other UI bits randomly disappear, you can usually fix it by reloading Discord:

  • Windows → Ctrl + R
  • Mac → Cmd + R
