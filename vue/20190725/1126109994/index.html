接下來打開畫面，就可以看到 card 的樣式出來了，那這就代表成功加載 bootstrap 了

為什麼要在 main.js import bootstrap?

如果只是單純使用 bootstrap 是可以不用修改 main.js，如果需要使用到有 jQuery 功能的 bootstrap 元件，那就要修改 main.js 唷~
