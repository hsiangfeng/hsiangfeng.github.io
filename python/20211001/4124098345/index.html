前言

前面我們也已經學了不少的知識點，而這邊也進入了鐵人賽的最後一天，所以就來試著寫一個簡單的 ithelp 鐵人賽爬蟲。

作業需求

一開始一樣先講一下我們要做什麼吧~

這邊我們主要會使用 Requests + BeautifulSoup 套件來實作一些東西，至於實作什麼就直接來看需求吧。

  • 爬取鐵人賽首頁獲取主題資訊，欄位必須有以下：
    • 主題網址
    • 主題標題
    • 參賽人數
  • 透過剛才的主題資訊撈取特別主題的參賽資料，欄位必須有以下：
    • id
    • 參賽者名稱
    • 鐵人賽名稱
    • 參賽者鐵人賽連結
    • 敲碗數
