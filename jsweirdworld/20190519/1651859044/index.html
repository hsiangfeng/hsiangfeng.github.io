#JavaScript #JavaScript е…Ёж”»з•Ҙ:е…ӢжңҚJSеҘҮжҖӘзҡ„йғЁеҲҶ

еүҚиЁҖ

йҖҷйӮҠе°ҮиҰҒдҫҶи¬ӣ JavaScript дёӯеҸҰдёҖеҖӢзү№еҲҘзҡ„й—ңйҚөеӯ—пјҢеңЁжҲ‘еҖ‘еҹ·иЎҢеҮҪж•ёжҷӮе°ұиҮӘеӢ•иЁӯе®ҡеҘҪзҡ„дёҖеҖӢй—ңйҚөеӯ— arguments еҸҠ ES6 зҡ„ spread еҸҲзЁұе…¶йӨҳеҸғж•ёгҖӮ

arguments

еңЁеҹ·иЎҢз’°еўғиў«е»әз«ӢжҷӮ JavaScript жңғжӣҝжҲ‘еҖ‘иЁӯе®ҡдёҖдәӣжқұиҘҝеғҸжҳҜи®Ҡж•ёз’°еўғдҫҶеҢ…дҪҸжҲ‘еҖ‘зҡ„и®Ҡж•ёгҖҒзөҰзҜ„еңҚйҸҲзҡ„еӨ–йғЁз’°еўғеҸғиҖғпјҢд»ҘеҸҠзү№ж®Ҡй—ңйҚөеӯ— thisпјҢеңЁжңҖеҫҢ JavaScript е…¶еҜҰйӮ„иЁӯе®ҡдәҶдёҖеҖӢзү№еҲҘзҡ„й—ңйҚөеӯ— argumentsгҖӮ

йӮЈ arguments жҳҜд»Җйәј?arguments еҢ…еҗ«дәҶжүҖжңүзҡ„еҖјпјҢжүҖжңүжҲ‘еҖ‘еӮіе…ҘеҮҪж•ёзҡ„еҸғиҖғзҡ„ж„ҸжҖқгҖӮ

arguments жҳҜеҸғж•ёзҡ„дёҖзЁ®жҰӮеҝөпјҢеҸӘжҳҜжҲ‘еҖ‘еӮіе…ҘеҮҪж•ёзҡ„и®Ҡж•ёзҡ„еҸҰдёҖзЁ®зЁұе‘јзҪ·дәҶпјҢжүҖд»ҘеҸҲеҸҜд»ҘзЁұзӮә parametersпјҢйҖҷжЁЈи¬ӣеҫҲжЁЎзіҠпјҢжүҖд»ҘеҫһзҜ„дҫӢзЁӢејҸзўјдҫҶзһӯи§Јд»ҖйәјжҳҜ arguments вҶ“

JavaScript иҲҮе…¶д»–иӘһиЁҖжңҖеӨ§дёҚеҗҢд№Ӣиҷ•е°ұжҳҜеҸҜд»ҘдёҚз”Ёеё¶е…Ҙд»»дҪ•еҸғж•ёд№ҹиғҪжӯЈеёёеҹ·иЎҢпјҢе„ҳз®ЎеҮҪж•ёдёҠйқўжңүйңҖиҰҒеё¶е…Ҙзҡ„еҸғж•ёпјҢдҪҶеҚ»д№ҹдёҚжңғи·ійҢҜиӘӨгҖӮ

еҸҜд»ҘзңӢеҲ°жҲ‘еҖ‘зҡ„еҮҪж•ёеҸғж•ёиў«иіҰдәҲй җиЁӯеҖј undefined * 3пјҢжүҖд»ҘжҲ‘еҖ‘д№ҹеҸҜд»ҘзҹҘйҒ“дёҖд»¶дәӢжғ…гҖӮ

еҮҪж•ёзҡ„еҸғж•ёпјҢе°ұжҳҜи®Ҡж•ёзҡ„дёҖзЁ®

йӮЈеҰӮжһңжҲ‘еҖ‘й–Ӣе§ӢеӮіе…ҘеҸғж•ёпјҢе®ғжңғз”ұжңҖе·ҰйӮҠиҮіеҸійӮҠдҫқеәҸеҠ е…Ҙ вҶ“

йӮЈжҲ‘еҖ‘дёҖжЁЈз№јзәҢеӮіе…ҘеҸғж•ё вҶ“

йҖҷжЁЈд»ЈиЎЁи‘—дёҖд»¶дәӢжғ…пјҢжҲ‘еҖ‘еҸҜд»ҘеҝҪз•ҘеӮіе…ҘеҸғж•ёпјҢеҸӘеӮіе…Ҙзү№е®ҡеҸғж•ёпјҢдҪҶжҳҜеңЁеҺҹз”ҹ JavaScript еҮҪж•ёеҸғж•ёжҳҜдёҚиғҪиіҰдәҲй җиЁӯеҖјеҸғж•ёпјҢжүҖд»ҘеҰӮжһңйңҖиҰҒдҪҝз”Ёй җиЁӯеҸғж•ёпјҢе°ұеҝ…й ҲдҪҝз”Ёд»ҘдёӢйҖҷеҖӢж–№жі• вҶ“

еӣһй ӯдҫҶи¬ӣдёҖдёӢеүӣеүӣеүҚйқўи¬ӣзҡ„ argumentsпјҢJavaScript еңЁеҹ·иЎҢз’°еўғе»әз«ӢжҷӮжӣҝжҲ‘еҖ‘жә–еӮҷеҘҪдәҶдёҖеҖӢ argumentsпјҢйӮЈ arguments жҳҜд»Җйәјй¬ј?и®“жҲ‘еҖ‘е°Үе®ғијёеҮәиҮі console зңӢзңӢ вҶ“

жҲ‘еҖ‘еҸҜд»ҘзҷјзҸҫдёҖд»¶дәӢжғ…пјҢarguments е…¶еҜҰжҳҜдёҖеҖӢйҷЈеҲ—пјҢиҖҢдё”еҢ…еҗ«дәҶжҲ‘еҖ‘жүҖжңүеӮіе…Ҙзҡ„еҸғж•ёеҖјпјҢдҪҶжҳҜиҰҒжіЁж„Ҹ arguments жҳҜдёҖеҖӢйЎһйҷЈеҲ—пјҢеӣ зӮәеҸғж•ёеңЁ JavaScript жҳҜдёҖеҖӢзү№еҲҘзҡ„жқұиҘҝпјҢжӯЈеӣ зӮәе®ғжҳҜйЎһйҷЈеҲ—пјҢжүҖд»Ҙе®ғеҸӘиғҪдҪҝз”ЁйғЁеҲҶйҷЈеҲ—зҡ„еҠҹиғҪгҖӮ

еҰӮжһңжІ’жңүеӮіе…Ҙд»»дҪ•еҸғж•ёпјҢжҲ‘еҖ‘еҸҜд»Ҙеҳ—и©ҰдҪҝз”Ё length дҫҶжӘўжҹҘйҷЈеҲ—й•·еәҰ вҶ“

з•¶з„¶д№ҹеҸҜд»ҘйҖҸйҒҺдҪҝз”Ё arguments еҸ–еҖј вҶ“

дҪҶжҳҜдҪ жңғзҷјзҸҫдёҖд»¶дәӢжғ…пјҢжҲ‘еҖ‘иЁӯзҪ®зҡ„ firstName = firstName || 'jack'; е°ұжІ’жңүж•Ҳжһңз…©иҖҢжҳҜ undefinedпјҢжүҖд»ҘеңЁдҪҝз”Ё arguments дҫҶзӣҙжҺҘеҸ–еҖјиҰҒе°ҸеҝғгҖӮ

spread (е…¶йӨҳеҸғж•ё)

spread жҳҜеңЁ ES6 еҠ е…Ҙзҡ„ж–°еҠҹиғҪпјҢи®“жҲ‘еҖ‘зңӢзңӢзҜ„дҫӢгҖӮ

жҲ‘еҖ‘еҸҜд»ҘзңӢеҲ°жңҖеҫҢдёҖиЎҢеҫҢйқўеӨҡдәҶе…©еҖӢеҸғж•ё greet('Mary','Tomy','us','123','test');пјҢйӮЈ spread и©ІжҖҺйәјз”Ё?з•¶жҲ‘еҖ‘дёҚзўәе®ҡеӮіе…Ҙзҡ„еҸғж•ёж•ёйҮҸжҷӮпјҢе°ұеҸҜд»ҘдҪҝз”Ё spread вҶ“

еҸӘиҰҒеҜ«е…Ҙ ... + еҸғж•ёеҗҚзЁұпјҢйҖҷжЁЈе°ұеҸҜд»ҘдәҶгҖӮ

еҸ–еҖјзҡ„ж–№ејҸе°ұдёҚеҶҚеӨҡиҝ°е°ұеғҸйҷЈеҲ—еҸ–еҖјдёҖжЁЈгҖӮ
