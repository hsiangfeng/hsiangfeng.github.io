invocation (тЄйТЋИтЉ╝тЈФ)

invocation тЈѕуе▒тЄйТЋИтЉ╝тЈФ№╝їжђЎТў»СИђтђІжЮътИИжЄЇУдЂуџёУДђт┐х№╝їТЅђС╗ЦуЋХС║║т«ХУгЏ invocation function or function invocation№╝їтЁХт»дт░▒Тў»тюеУгЏтЪиУАїжђЎтђІтЄйТЋИ№╝їУђїТѕЉтђЉТў»жђЈжЂј**ТІгУЎЪ()**СЙєтЪиУАїтЄйТЋИсђѓ

Уф▓уеІТюЅТЈљСЙЏС║єСИђтђІтЪ║уцју»ёСЙІ№╝їт╣ФтіЕТѕЉтђЉуљєУДБУДђт┐х РєЊ

жђЎТЎѓтђЎТѕЉтђЉтЈ»С╗ЦТђЮУђЃСИђСИІ JavaScript Тў»тдѓСйЋжЂІСйюуџёсђѓ

  1. ждќтЁѕтЁетЪЪтЪиУАїуњ░тбЃТюЃУбФтЁѕт╗║уФІ№╝їС╣Ът░▒Тў» window
  2. УфъТ│ЋУДБТъљтЎежќІтДІтѕєТъљТѕЉтђЉуџёуеІт╝ЈтЁДт«╣ f-u-c-t-Рђд.(жђљтГЌУДБТъљ)
  3. уиеУГ»тЎежќІтДІт░ЄТѕЉтђЉТЅђТњ░т»ФуџёуеІт╝ЈУйЅТЈЏжЏ╗УЁдтЈ»С╗ЦуюІТЄѓтЙЌТїЄС╗ц
  4. жќІтДІтЅхжђасђїтЁетЪЪтЪиУАїуњ░тбЃсђЇ№╝їжђЎТЎѓтђЎТюЃт╗║уФІ window тЈі this уГЅтЁетЪЪуЅЕС╗Х
  5. тєЇСЙєт░▒жђ▓тЁЦтЅхжђажџјТ«х (Hoisting) жќІтДІтЅхжђаТѕЉтђЉуџёУ«іТЋИтЈітЄйТЋИ

жѓБтЄйТЋИжЃйТюЃтёфтЁѕУбФтё▓тГўтюеУеўТєХжФћСИГ№╝їуЏ┤тѕ░ТѕЉтђЉтЉ╝тЈФжђЎС║ЏтЄйТЋИ№╝їт«ЃТЅЇТюЃтЪиУАїтЄйТЋИСИГуџёуеІт╝Ј№╝їТЅђС╗ЦтюетЅЇжЮбуџёу»ёСЙІ РєЊ

жђЎС║ЏТЎѓтђЎтЄйТЋИтЈфТў»тё▓тГўтюеУеўТєХжФћСИГУђїти▓№╝їуЏ┤тѕ░т«ЃтЙђСИІжЂЄтѕ░ТѕЉтђЉТЅђТњ░т»Фуџё invocation№╝їС╣Ът░▒Тў» a() ТЎѓ№╝їТЅЇТюЃжќІтДІтЪиУАїтЄйТЋИСИдтЪиУАїУБАжЮбуџёуеІт╝Јуб╝№╝їжѓБжђЎжѓіуеЇтЙ«тЂюТГбСИђСИІ№╝їуЋХтЪиУАї a() ТЎѓтђЎ№╝їтЁХт»дућбућЪС║єСИђтђІтЙѕтЦЄтдЎуџёС║ІТЃЁ№╝їТќ░уџётЪиУАїуњ░тбЃУбФтЅхжђаС║є№╝їУђїжђЎжЂјуеІт░▒Тў»ТЅђУгѓуџётЪиУАїтаєсђѓ

УђїтЪиУАїтаєт░▒Тў»СИђт▒цСИђт▒цуќіСИітј╗уџё№╝їжѓБУф░тюеТюђСИіжЮбУф░т░▒Тў»ТГБтюетЪиУАїСИГуџёТЮ▒УЦ┐№╝їТЅђС╗ЦТѕЉтђЉТ»ЈТгАтюетЉ╝тЈФтЄйТЋИТЎѓ№╝їжЃйТў»тюет╗║уФІТќ░уџётЪиУАїуњ░тбЃ№╝їуёХтЙїТћЙжђ▓тЪиУАїтаєСИГсђѓ

УђїжђЎтђІТќ░уџётЪиУАїуњ░тбЃтЙѕуЅ╣тѕЦ№╝їт«ЃТюЅт▒гТќ╝УЄфти▒уџётЁетЪЪтЪиУАїуњ░тбЃсђЂУеўТєХжФћС╗ЦтЈітЄйТЋИуГЅ№╝їУђїт«ЃС╣ЪТюЃТГиуХЊтЅхжђасђЂТЈљтЇЄуГЅжџјТ«х№╝їуёХтЙїжђљУАїжђљтГЌуџётЪиУАїуеІт╝Јуб╝сђѓ

ТЅђС╗ЦуЋХтЪиУАї b() уџёТЎѓтђЎ a() т░▒ТюЃтЂюТГбжЂІСйю№╝їуёХтЙїтЅхжђаТќ░уџётЪиУАїуњ░тбЃ№╝їТЅђС╗Цућ▒ТГцтЈ»уЪЦТ»ЈСИђтђІтЄйТЋИжЃйТюЃт╗║уФІСИђтђІТќ░уџётЪиУАїуњ░тбЃСИдСИћТГиуХЊтЅхжђауГЅуГЅжџјТ«х№╝їУђїуеІт╝Јуб╝уџётЪиУАїжаєт║ЈС╣ЪСИЇТюЃтЈЌтѕ░тЪиУАїтаєуџётЋЈжАїУђїтй▒жЪ┐сђѓ

СйєУІЦтдѓТъюжђЎТеБтГљтЉб?

ждќтЁѕТѕЉтђЉТюЃуЪЦжЂЊ a() тЄйТЋИТюЃУбФтЉ╝тЈФУђї a() т░▒ТюЃУ«іТѕљуЏ«тЅЇТГБтюетЪиУАїСИГуџёуеІт╝Јсђѓ

СйєТў»ждќтЁѕТюђтЙїСИђУАїуџё var d тюе a() уџёТюђтЙїСИђУАї№╝їтЏауѓ║тЄйТЋИ a() жѓёТ▓њТюЅтЪиУАїт«їуЋб№╝їТЅђС╗ЦСИЇТюЃУбФт╗║уФІ№╝їСИ╗УдЂтјЪтЏаТў» JavaScript Тў»тљїТГЦтЪиУАї№╝їСИдСИђТгАтЪиУАїСИђУАїСИђтђІТїЄС╗ц№╝їТЅђС╗Цт«Ѓт┐ЁжаѕуГЅтЄйТЋИ a() тЪиУАїт«їуЋбТЅЇТюЃт╗║уФІсђѓ

ТјЦСИІСЙє a() тЄйТЋИтЪиУАїжЂјуеІСИГ№╝їт«ЃТюЃтЉ╝тЈФ b()№╝їуёХтЙї b() т░▒ТюЃжќІтДІт╗║уФІУЄфти▒уџётЪиУАїуњ░тбЃСИдСИћТѕљуѓ║ТюђСИіжЮбуџётЪиУАїСИГуџёуеІт╝Ј№╝їуёХтЙїтюетЪиУАї var d№╝їуЋХ b() тЪиУАїт«їуЋбтЙї b т░▒ТюЃжЄІТћЙТјЅ(С╣Ът░▒ТюЃжЏбжќІтЪиУАїтає)№╝їжђЎТЎѓтђЎт░▒ТюЃтЈфтЅЕСИІ a() жѓётюетЪиУАїтаєСИГ№╝їтЏауѓ║т«ЃжѓёУдЂтЅхжђаУ«іТЋИ var c№╝їжѓБт«Ѓт╗║уФІт«їуЋбС╣ІтЙїтЉб?С╣ЪТў»СИђТеБТюЃУбФжЄІТћЙТјЅ№╝їжђЎТЎѓтђЎТЅЇТюЃтЪиУАїТюђтЙїСИђУАїуџё var d;сђѓ

ТЅђС╗ЦТѕЉтђЉтЈ»С╗ЦС║єУДБтѕ░Т»ЈтЪиУАїСИђтђІтЄйТЋИт░▒ТюЃт╗║уФІСИђтђІТќ░уџётЪиУАїуњ░тбЃ№╝їуёХтЙїжђљУАїСИђУАїжђљтЈЦжђљтГЌуџётљїТГЦтЪиУАї№╝їуЏ┤тѕ░т«ЃтЪиУАїт«їуЋбсђѓ
