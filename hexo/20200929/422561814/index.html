е®үиЈқ minos дё»йЎҢ

йӣ–з„¶е®үиЈқдё»йЎҢзҡ„йҒҺзЁӢеҹәжң¬дёҠеӨ§еҗҢе°Ҹз•°пјҢдҪҶжҳҜйӮ„жҳҜжңүдёҖдәӣдё»йЎҢеӣ зӮәеңЁиЁӯзҪ®дёҠжҜ”ијғзү№ж®ҠпјҢйӮ„жҳҜжңғиё©йӣ·е”·~

жүҖд»Ҙеә•дёӢе°ұзӣҙжҺҘжҸҗдҫӣдё»йЎҢзҡ„жҢҮд»Ө

йҖҷйӮҠдёҖжЁЈзҡ„еӢ•дҪңжҳҜеҲ° hexo/_config.yml дҝ®ж”№ theme еұ¬жҖ§жҲҗ minos

з„¶еҫҢз”ұж–ји©Ідё»йЎҢзҡ„иЁӯзҪ®ж–№ејҸиҲҮ Hueman зӣёеҗҢпјҢminos еҸӘжҸҗдҫӣзөҰдҪ  _config.yml.example дёҰжІ’жңү themes/_config.ymlпјҢеӣ жӯӨйҖҷйӮҠиҰҒијёе…Ҙд»ҘдёӢжҢҮд»ӨдҫҶжӢ·иІқдёҖд»ҪеҮәдҫҶ

жҺҘдёӢдҫҶдҪ еҸҜиғҪжңғзӣҙжҺҘијёе…Ҙ hexo s дҫҶжЁЎж“¬дјәжңҚеҷЁпјҢдҪҶжҳҜйҖҷйӮҠдҪ дёҖе®ҡжңғеҮәзҸҫйҢҜиӘӨ

йҖҷжҳҜеӣ зӮә minos дё»йЎҢйӮ„еҝ…й ҲйЎҚеӨ–е®үиЈқдёҖдәӣеҘ—д»¶жүҚиғҪжӯЈеёёйҒӢдҪңпјҢеңЁйҖҷйӮҠиҰҒијёе…Ҙд»ҘдёӢжҢҮд»Ө

жӯӨжҷӮжүҚиғҪеӨ жӯЈеёёйҒӢдҪңжЁЎж“¬дјәжңҚеҷЁе”·

minos иЁӯзҪ®жӘ”жЎҲ

minos д№ҹжҳҜзӣёе°ҚжҜ”ијғзҹӯпјҢеӣ жӯӨйҖҷйӮҠе°ұдёҚеҶҚд»Ӣзҙ№пјҢдё»иҰҒжңғд»Ӣзҙ№зҡ„ең°ж–№жҳҜдҝ®ж”№иӘһиЁҖзҡ„йғЁеҲҶпјҢminos жңғеҹәж–јдҪ зҡ„ theme/_config.yml дҫҶеҲҮжҸӣиӘһзі»пјҢеӣ жӯӨйҖҷйӮҠеҰӮжһңдҪ еёҢжңӣз•«йқўе‘ҲзҸҫзҡ„жҳҜз№Ғй«”дёӯж–ҮпјҢйӮЈйәје°ұиҰҒе°ҮеҺҹжң¬зҡ„ theme/_config.yml дҝ®ж”№жҲҗ theme/_config.zh-tw.ymlгҖӮ

з•¶з„¶ minos йӮ„жңүдёҖеҖӢзү№иүІе°ұжҳҜж”ҜжҸҙеӨҡеңӢиӘһзі»пјҢеӣ жӯӨдҪ еҸҜд»Ҙдҫқз…§зӣёй—ңиӘһзі»е»әз«Ӣеә•дёӢжӘ”жЎҲдҫқеәҸијёе…Ҙд»ҘдёӢжҢҮд»Өе»әз«ӢеӨҡеңӢиӘһзі»жӘ”жЎҲ

еҹәжң¬дёҠ themes/_config.en.yml еҸҜд»ҘдёҚз”ЁеӢ•пјҢз•ўз«ҹжң¬иә«е°ұжҳҜиӢұж–ҮпјҢдё»иҰҒжңғж”№зҡ„жҳҜ themes/_config.zh-tw.ymlпјҢжү“й–Ӣ themes/_config.zh-tw.yml д№ӢеҫҢжҗңе°Ӣ # Navigation bar menu links. дёҰе°Үе…§е®№дҝ®ж”№жҲҗд»ҘдёӢ

жҺҘдёӢдҫҶз”ұж–јжҲ‘еҖ‘жІ’жңү /zh-tw/ зҡ„иіҮж–ҷеӨҫиҲҮдёӢйқўзӣёй—ңй ҒйқўпјҢжүҖд»ҘиЁҳеҫ—иҰҒеңЁ _posts дёӢе»әз«ӢиіҮж–ҷеӨҫеҸ« zh-twпјҢжҺҘдёӢдҫҶеңЁе»әз«Ӣзӣёй—ңй ҒйқўеҚіеҸҜ

еҹәжң¬дёҠйҖҷжЁЈеӯҗдҪ еҹәзӨҺзҡ„еӨҡеңӢиӘһзі»йғЁиҗҪж је°ұеҘҪдәҶгҖӮ

иҖҢйҖҷдёҖзҜҮдё»иҰҒзҡ„дҪҝз”ЁиҖ…жҳҜвҖҰHuliпјҲиғЎз«Ӣпјү дҪ д№ҹеҸҜд»ҘйҖӣйҒҺзңӢйҒҺеҶҚжұәе®ҡжҳҜеҗҰдҪҝз”Ёи©Ідё»йЎҢе”·~

minos е°Ҹе•ҸйЎҢ

жңҖеҫҢйҖҷйӮҠдҪ еҸҜиғҪжңғжіЁж„ҸеҲ°ж–Үз« жЁҷйЎҢдёӢзҡ„жҷӮй–“жҳҜз°Ўй«”еӯ—гҖҢx дёӘжңҲеүҚ x еҲҶй’ҹ и®Җе®Ң (зҙ„ xxx еӯ—)гҖҚпјҢеңЁйҖҷйӮҠзҡ„дҝ®жӯЈж–№жі•жҳҜдҝ®ж”№дҪҚж–ј minos/scripts/99_content.js з„¶еҫҢжүҫеҲ° const MOMENTJS_SUPPORTED_LANGUAGES йҖҷеҖӢи®Ҡж•ёпјҢжҺҘдёӢдҫҶе°Ү zh-tw ж”№жҲҗ zh-TW е°ұеҸҜд»Ҙи§Јжұәеӣү~

иҮіж–јеҺҹеӣ жҳҜд»ҖйәјжҲ‘д№ҹдёҚжё…жҘҡ(жІ’ж·ұе…Ҙз ”з©¶)пјҢжҮүи©ІжҳҜиӘһзі»еҲӨж–·дёҠе•ҸйЎҢпјҢе®ғдё»иҰҒжҳҜжҠ“ hexo/_config.yml дёӯзҡ„ language еұ¬жҖ§дҫҶиҫЁеҲҘиӘһзі»пјҢдҪҶжҳҜйҖҷйӮҠе•ҸйЎҢи§Јжұәе°ұеҘҪе“©~
