Google Search Console

Google Search Console еүҚиә«еҸ«еҒҡ Google Webmaster ToolпјҢз°Ўе–®дҫҶи¬ӣе°ұжҳҜе°Ҳй–Җз®ЎзҗҶдҪ з¶Із«ҷеңЁжҗңе°Ӣеј•ж“ҺдёҠзҡ„ SEO иЎЁзҸҫпјҢе…¶дёӯеҰӮжһңдҪ жғізҹҘйҒ“дҪ зҡ„з¶Із«ҷе“ӘдәӣжҳҜзҶұй–Җй—ңйҚөеӯ—пјҢйӮЈйәје°ұдёҚиғҪйҢҜйҒҺйҖҷеҖӢе·Ҙе…·гҖӮ

еңЁдёҖй–Ӣе§ӢйҖІе…Ҙ Google Search Console д№ӢеүҚдҪ еҝ…й Ҳе…ҲиЁӯе®ҡй©—иӯүиҮӘе·ұзҡ„з¶Із«ҷ

(йҖҡеёёз¬¬дёҖж¬ЎйҖІдҫҶжңғи·іеҮәи©ІиҰ–зӘ—пјҢеҰӮжһңжІ’жңүи·іеҮәдҫҶзҡ„и©ұпјҢеҸҜд»Ҙй»һдёҖдёӢе·ҰдёҠз¶Іеҹҹзҡ„ menu е°ұеҸҜд»ҘзңӢеҲ°гҖҢж–°еўһиіҮжәҗгҖҚзҡ„жҢүйҲ•дәҶгҖӮ)

еңЁйҖҷйӮҠжҲ‘е»әиӯ°дҪҝз”ЁгҖҢз¶ІеқҖеүҚзҪ®еӯ—е…ғгҖҚдҫҶй©—иӯүз¶Із«ҷдҪҝз”ЁиҖ…пјҢжҺҘдёӢдҫҶдҪ жңғзңӢеҲ°еҗ„зЁ®й©—иӯүж–№ејҸпјҢеҰӮжһңдҪ жң¬иә«жңүдҪҝз”Ё GA зҡ„и©ұеҸҜд»ҘдҪҝз”Ё GA й©—иӯүеҚіеҸҜпјҢеңЁйҖҷйӮҠз”ұж–јжҲ‘еҖ‘жҳҜз¬¬дёҖж¬ЎдҪҝз”ЁпјҢжүҖд»Ҙе°ұдҪҝз”Ёе®ғжҺЁи–Ұзҡ„ HTML жӘ”жЎҲй©—иӯү

дёӢијүеҘҪзҡ„ .html жӘ”жЎҲпјҢеҸӘйңҖиҰҒж”ҫеҲ° hexo/source еә•дёӢеҚіеҸҜпјҢдҪҶжҳҜеңЁйӮЈд№ӢеүҚжҲ‘еҖ‘еҝ…й Ҳдҝ®ж”№дёҖдёӢе…§е®№пјҢеҗҰеүҮдҪ еңЁй©—иӯүдёҠжңғеҮәзҸҫйҢҜиӘӨпјҢиҲүдҫӢдҫҶи¬ӣжҲ‘зҡ„е…§е®№жҳҜ google-site-verification: google03c774646bdb5e4a.htmlпјҢйӮЈйәје°ұиҰҒж”№жҲҗд»ҘдёӢ

дҝ®ж”№е®Ңз•ўд№ӢеҫҢжүҚиғҪеӨ ж”ҫеҲ° source е”·~

йҖҷйӮҠиҰҒжіЁж„ҸеҰӮжһңдҪ жІ’жңүеҠ дёҠ layout: false жҳҜжңғеҮәзҸҫз„Ўжі•й©—иӯүзҡ„е•ҸйЎҢе”·

жҺҘдёӢдҫҶе°ұжҳҜзӣҙжҺҘйғЁзҪІ hexo еҲ° GitHub PagesпјҢзӮәдәҶйҒҝе…Қеҗ„дҪҚ йҮ‘йӯҡи…Ұ еҝҳиЁҳпјҢжүҖд»ҘжҲ‘еҖ‘иҰҒијёе…Ҙ hexo d -g дҫҶе°Үзӣ®еүҚ Hexo йғЁзҪІеҲ°жҲ‘еҖ‘зҡ„з©әй–“еҺ»пјҢжңҖеҫҢеҶҚеӣһдҫҶ Google Search Console жҢүдёӢгҖҢй©—иӯүгҖҚжҢүйҲ•еҚіеҸҜгҖӮ

еӨ§жҰӮйқңзҪ®е…©дёүеӨ© Google Search Console е°ұжңғеҲҶжһҗеҮәдҪ зҡ„йғЁиҗҪж јжңүе“ӘдәӣзҶұй–Җй—ңйҚөеӯ—д»ҘеҸҠжңүе“Әдәӣең°ж–№еҸҜд»Ҙе„ӘеҢ–пјҢеҹәжң¬дёҠеҸӘиҰҒйҒөеҫӘе®ғжҸҗдҫӣзҡ„ж–№еҗ‘еҺ»иӘҝж•ҙдҪ зҡ„ SEO е°ұжңғдёҠеҚҮзӣёз•¶еӨҡгҖӮ

йҷ„её¶дёҖжҸҗпјҢGoogle й©—иӯүзҡ„иіҮжәҗ .html и«ӢдёҚиҰҒз§»йҷӨпјҢеӣ зӮәжҜҸдёҖж¬Ў Google йғҪжңғй©—иӯүи©ІиіҮжәҗе”·~

жҸҗдәӨз¶ІеқҖзөҰ Google зҙўеј•

з”ұж–ј Google Search Console еҠҹиғҪйқһеёёеј·еӨ§д№ҹйқһеёёеӨҡзҙ°зҜҖеҸҜд»ҘзңӢпјҢйҖҷйӮҠжҲ‘е°ұдёҚдёҖдёҖд»Ӣзҙ№пјҢдҪҶжҳҜйҖҷйӮҠжңҖдё»иҰҒдёҖе®ҡиҰҒд»Ӣзҙ№дёҖеҖӢең°ж–№пјҢд№ҹе°ұжҳҜ Sitemap еҠҹиғҪпјҢд»ҘеҫҖжҲ‘еҖ‘дёҖеҖӢж–°зҡ„з«ҷеүӣдёҠз·ҡжҷӮпјҢеҰӮжһңжІ’жңүдёҖдәӣз¶Із«ҷдә’зӣёй—ңиҒҜпјҢйӮЈйәје°ұеҫҲйӣЈиў«зҲ¬иҹІзөҰжҗңе°ӢеҲ°пјҢжүҖд»ҘйҖҷжҷӮеҖҷжҲ‘еҖ‘е°ұеҝ…й ҲжӣҝиҮӘе·ұзҡ„ Hexo е®үиЈқдёҖеҖӢеҘ—д»¶жҳҜе°Ҳй–Җз”ҹжҲҗ sitemap.xmlпјҢиҖҢйҖҷеҖӢжӘ”жЎҲжҳҜе°Ҳй–ҖжҸҗдәӨзөҰжҗңе°Ӣеј•ж“ҺзҲ¬иҹІдҪҝз”ЁпјҢзҷҪи©ұж–Үзҡ„иӘӘжі•е°ұжҳҜгҖҢеҳҝпјҒжҲ‘зҡ„йғЁиҗҪж јз¶ІеқҖеңЁйҖҷиЈЎпјҢжңүз©әдҫҶе№«жҲ‘ж”¶йҢ„пјҒгҖҚзҡ„жҰӮеҝөгҖӮ

еӣ жӯӨжҲ‘еҖ‘йңҖиҰҒе®үиЈқеҸҜд»ҘиҮӘеӢ•з”ҹжҲҗ sitemap.xml зҡ„еҘ—д»¶пјҢеңЁйҖҷйӮҠжҲ‘жҸҗдҫӣд»ҘдёӢжҢҮд»Өж–№дҫҝдҪ еҝ«йҖҹиӨҮиЈҪ

жҺҘдёӢдҫҶеңЁијёе…ҘдёҖж¬ЎйғЁзҪІзҡ„жҢҮд»Ө(йҖҷеҖӢжҷӮеҖҷ Hexo жңғиҮӘеӢ•з”ҹжҲҗдёҖеҖӢжӘ”жЎҲеҸ«еҒҡ sitemap.xmlпјҢе®Ңе…ЁжҳҜиҮӘеӢ•зҡ„е”·пјҒ)

йғЁзҪІе®Ңз•ўд№ӢеҫҢзӣҙжҺҘеҲ° Google Search Console дёҠзҡ„гҖҢSitemapгҖҚеҠҹиғҪй ҒйқўпјҢдёҰеңЁгҖҢж–°еўһ SitemapгҖҚж¬„дҪҚијёе…Ҙ sitemap.xml з„¶еҫҢжҸҗдәӨеҚіеҸҜ

йҖҷжЁЈеӯҗд»ҘеҫҢ Google зҡ„зҲ¬иҹІе°ұжңғе®ҡжңҹеҺ»зңӢ sitemap.xml жӘ”жЎҲжҳҜеҗҰжңүжӣҙж–°пјҢиӢҘжңүжӣҙж–°е®ғе°ұжңғеҺ»зҲ¬дҪ зҡ„йғЁиҗҪж јдёҰеҠ е…ҘеҲ°жҗңе°Ӣеј•ж“Һдёӯеӣү~

йҖҡеёёйғЁиҗҪж јиҰҒиў«жҗңе°Ӣеј•ж“Һж”¶е…ҘпјҢеӨ§жҰӮдҝқе®Ҳдј°иЁҲиҮіе°‘иҰҒ 7 ~ 14 еӨ©дёҚзӯүеҗ§пјҹжҲ‘д№ҹжңүйҒҮйҒҺз•¶еӨ©е°ұжңүж”¶йҢ„зҡ„зӢҖжіҒпјҢеӣ жӯӨе®Ңе…ЁиҰҒзңӢзҲ¬иҹІзҡ„зӢҖжіҒзӮәдё»е“©гҖӮ

йӮЈйәјйҖҷдёҖзҜҮе°ұе…ҲеҲ°йҖҷйӮҠзөҗжқҹпјҢжҺҘдёӢдҫҶе°ұжҳҜжңҖеҫҢдёҖзҜҮ SEO зҡ„д»Ӣзҙ№дәҶгҖӮ
