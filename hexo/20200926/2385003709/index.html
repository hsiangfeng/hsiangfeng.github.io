主題的挑選

挑選主題一直都是許多人煩惱的地方，雖然 Hexo 官方網站的導覽列有一個主題的選項，數量約有上百個主題，看似非常多但實際要注意的細節反而更多，而且挑選主題就跟玩遊戲取暱稱一樣，想了老半天還是不知道要取什麼暱稱好

當發現自己花太多時間時，就會開始隨便取一些奇怪的名字，例如…

拿佛經打耶穌、靈車甩尾棺材飛、蒙奇D能兒之類等等的詭異暱稱，然後用了下去之後才發現自己這個遊戲暱稱腦殘到一個極點，這時候只會有兩條路

  • 花錢買改名卡
  • 砍掉重練

當然選 Hexo 的主題也是一樣的，如果你沒有慎選一個主題，除了 SEO 要重來之外，你也有可能面臨主題要重新挑選，所以在挑擇主題時，千萬不要像玩遊戲一樣亂取暱稱，尤其是當你取一個不得了的暱稱，像是 下麵給我吃 這種爛暱稱，然後想去找一個網婆，我相信哪一天大概你就會收到私訊「腦公，我明天要當兵了~」QQ

挑選主題的技巧

基本上在挑選主題時只有兩個重點

  • 主題是否有定期在維護
  • 社群討論度夠不夠

首先先來講講主題是否有定期在維護這一點，主要原因在於 Hexo 4 在編譯效能上是比 Hexo 3 好的，如果主題若沒有定期在維護，那麼就有可能該主題沒有辦法被新版的 Hexo 編譯，這樣子的下場就是只有被淘汰的份，除非你功力非常高深莫測可以自幹改主題。

接下來關於社群討論度的問題，如果你找了一個比較冷門的主題，那麼當你遇到任何問題其實都會格外難處理，儘管是一個微不足道的小問題，也有可能讓你跟 Google 小姐聊天好幾天。

以上這兩個狀況其實如果你本身有一定實力，並且研究過 Hexo 主題開發，那麼使用什麼主題對於你自己來講並沒有太大差異，反正你自己改就好了，但是如果你是跟我一樣很懶的話，那麼就會建議你挑選一些比較熱門的主題會比較好。

通常來講，比較多人使用的主題 Issues 數量都普遍滿多的，畢竟越多人使用就越會發生一些奇怪的問題(？)舉例來講 Hexo 中較熱門的主題就非 NexT 莫屬（後面會介紹），它的 Issues 數量就滿多的

除此之外比較熱門的主題在 GitHub 的星星數量也會比較多

透過這幾個方向去挑選主題，我相信你應該比較不會挑選到一些很雷的主題，否則當主題不維護或者太冷門，遇到問題時是真的會很痛苦唷。
